# 퀀트 서비스 지표 및 점수 정의서 (Enhanced)

이 문서는 `quant_collector_enhanced.py`에서 수집하는 데이터와 `quant_screener.py`에서 계산하는 투자 지표들의 상세 정의 및 활용 가이드를 제공합니다. 웹 대시보드 상세 모달에 표시되는 모든 지표를 그룹별로 완벽하게 정리했습니다.

---

## 1. 가격 & 밸류에이션 (Price & Valuation)
기업의 규모와 내재 가치 대비 현재 주가 수준을 판단하는 지표들입니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **현재가** | 가장 최근 거래일의 종가 | - |
| **시가총액** | 현재가 × 상장주식수 | 기업의 실제 시장 가치 (규모 판별 기준) |
| **PER** | 주가수익비율 = 시가총액 / TTM 순이익 | 낮을수록 저평가. (PER_이상: 0.5미만 또는 500초과 시 플래그) |
| **PBR** | 주가순자산비율 = 시가총액 / 최근 자본총계 | 1 미만이면 청산가치보다 싼 상태. |
| **PSR** | 주가매출비율 = 시가총액 / TTM 매출액 | 적자 기업이나 성장주 평가 시 유용. |
| **PEG** | PER / 순이익 CAGR (최대 100% 캡) | 0.5 이하 강력 매수, 1.0 적정, 1.5 이상 고평가 (성장성 대비 저평가). |
| **EPS** | 주당순이익 = TTM 순이익 / 상장주식수 | 주식 1주가 벌어들인 순이익. |
| **BPS** | 주당순자산 = 자본총계 / 상장주식수 | 주식 1주당 할당되는 순자산. |
| **이익수익률%** | (TTM 순이익 / 시가총액) × 100 | 주식을 채권으로 볼 때의 기대수익률 (국채 금리와 비교). |
| **S-RIM 적정가** | BPS + BPS × (ROE - Ke) × ω / (1 + Ke - ω) | 기업의 초과이익(ROE > 요구수익률 9.0%) 창출 능력을 반영한 내재가치. |
| **괴리율%** | (S-RIM 적정가 - 현재가) / 현재가 × 100 | 양수(+)면 적정주가 대비 저평가되어 상승 여력이 있음. |

## 2. 수익성 (Profitability)
자본과 자산으로 돈을 얼마나 효율적으로 벌고, 그 이익이 진짜 현금인지 확인합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **ROE%** | (TTM 순이익 / 자본총계) × 100 | 10% 이상 양호, 20% 이상은 경제적 해자를 보유한 초우량 기업. |
| **영업이익률%** | 최신 연간 기준 (영업이익 / 매출액) × 100 | 기업의 본업 경쟁력 및 마진율. |
| **FCF수익률%** | (TTM FCF / 시가총액) × 100 | 시가총액 대비 주주가 가져갈 수 있는 잉여현금의 비율 (3% 이상 양호). |
| **이익품질** | O/X (TTM 영업CF > TTM 순이익 이면 O) | O이면 장부상 이익이 실제 현금으로 잘 들어오고 있음을 의미. |
| **현금전환율%** | (TTM 영업CF / TTM 순이익) × 100 | 순이익이 현금으로 전환되는 비율 (100% 이상일수록 이익의 질이 우수). |
| **CAPEX비율%** | (TTM CAPEX / TTM 영업CF) × 100 | 번 현금 중 설비투자에 쓴 비율 (너무 높으면 현금이 메마를 수 있음). |

---

## 3. 성장성 (Growth)
매출과 이익이 과거부터 어떻게 성장해 왔는지, 최근 분기에 어떤 추세인지 확인합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **매출CAGR%** | 최근 n년간 매출액 연평균 성장률 | 외형의 장기 성장 속도. 10~15% 이상이면 고성장. |
| **OP CAGR%** | 최근 n년간 영업이익 연평균 성장률 | 본업 이익의 장기 성장 속도. |
| **NP CAGR%** | 최근 n년간 순이익 연평균 성장률 | 최종 이익의 장기 성장 속도. |
| **Q 매출YoY%** | 최근 분기 매출액 전년 동기 대비 증감률 | 가장 최근의 외형 성장 모멘텀. |
| **Q OP YoY%** | 최근 분기 영업이익 전년 동기 대비 증감률 | 가장 최근의 핵심 이익 성장 모멘텀. |

---

## 4. 재무건전성 (Stability & F-Score)
기업이 빚을 갚을 능력이 있는지, 재무 리스크가 없는지 종합적으로 평가합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **F-Score** | 피오트로스키 F-스코어 종합 점수 (0~9) | 7~9점이면 우량(Strong Buy), 0~3점이면 부실/위험. |
| **F1 수익성** | 당기 순이익이 양수(>0)이면 O | |
| **F2 영업CF** | 영업활동현금흐름이 양수(>0)이면 O | |
| **F3 ROA개선** | 당기 ROA > 전년 ROA 이면 O | |
| **F4 이익품질** | 영업CF > 순이익 이면 O | |
| **F5 레버리지** | 당기 부채비율 < 전년 부채비율 이면 O | |
| **F6 유동성** | 당기 유동비율 > 전년 유동비율 이면 O | |
| **F7 희석없음** | 신규 주식 발행(유상증자 등)이 없으면 O | |
| **F8 매출총이익률** | 당기 GPM > 전년 GPM 이면 O | |
| **F9 자산회전율** | 당기 자산회전율 > 전년 자산회전율 이면 O | |
| **부채비율%** | (부채총계 / 자본총계) × 100 | 100% 이하 안정, 200% 초과 주의. |
| **부채상환능력** | TTM 영업CF / 부채총계 | 1년 동안 번 현금으로 전체 부채를 갚을 수 있는 역량. |
| **이자보상배율** | TTM 영업이익 / 이자비용 | 1 미만이면 번 돈으로 이자도 못 갚는 상태. |

## 5. 배당 (Dividend)
주주환원 정책의 강도와 배당 성장을 확인합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **배당수익률%** | (최근 주당배당금 / 현재가) × 100 | 은행 금리와 비교, 주가 하방 경직성 확보의 핵심 지표. |
| **DPS** | 주당배당금(원) | 1주당 지급된 실제 배당금. |
| **DPS CAGR%** | 최근 n년간 주당배당금 연평균 성장률 | 배당성장주 필터링의 핵심 지표. |
| **배당연속증가** | 배당금이 연속으로 전년 대비 증가한 횟수(년) | - |
| **수익동반증가** | 순이익 2년 연속 성장 & 배당 1년 이상 성장 여부 | 이익 성장과 주주환원이 같이 이루어지고 있는 이상적인 상태. |

## 6. 기술적 지표 (Technical)
주가 추세, 변동성, 시장의 관심을 파악합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **52주 최고대비%** | (현재가 / 52주 최고가 - 1) × 100 | 신고가 돌파 직전이거나 낙폭이 과대한지 파악. (0%면 신고가) |
| **52주 최저대비%** | (현재가 / 52주 최저가 - 1) × 100 | 바닥에서 얼마나 올라왔는지 확인. |
| **MA20%** | 현재가 / 20일 이동평균선 이격도 | 단기 과열 여부 확인 (보통 5% 이상 과열 우려). |
| **MA60%** | 현재가 / 60일 이동평균선 이격도 | 중기 추세 확인. |
| **RSI(14)** | 14일 상대강도지수 (0~100) | 30 이하 과매도(바닥권), 70 이상 과매수(단기 고점). |
| **거래대금(20일평균)** | 최근 20일 거래대금 평균 | 시장의 관심도 측정. 주도주는 거래대금이 풍부함. |
| **거래대금 증감%** | 최근 5일 평균 vs 20일 평균 거래대금 비교 | 최근 거래량이 폭발하며 시장의 관심이 집중되었는지 판단. |
| **변동성(60일)%** | 최근 60일 주가 수익률의 연율화 표준편차 | 수치가 낮을수록 주가 흐름이 안정적이고 수렴 중임을 의미. |

---

## 7. 모멘텀 & 상대강도 (Momentum & Relative Strength)
전체 시장(KOSPI/KOSDAQ) 대비 얼마나 아웃퍼폼하고 있는지 확인합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **RS(60d)** | 60일 종목 수익률 - 지수 수익률 | 단기(3개월) 시장 대비 초과 수익률. |
| **RS(120d)** | 120일 종목 수익률 - 지수 수익률 | 중기(6개월) 시장 대비 초과 수익률. |
| **Composite RS** | 60일(40%), 120일(30%), 250일(30%) RS의 가중합 | 기간별 강도를 복합적으로 평가한 절대 수치. |
| **RS 등급** | Composite RS의 전 종목 퍼센타일 랭크 (0~100) | 80 이상이면 시장 상위 20%의 초강력 주도주(윌리엄 오닐 전략). |

---

## 8. 수급 & 매집 (Supply & VCP)
기관 및 외국인 투자자의 자금 유입과 매집 패턴을 분석합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **수급강도** | 최근 20일 (외인순매수+기관순매수) / 시총 × 100 | 양수(+)면 메이저 세력이 유의미하게 물량을 매집 중이라는 뜻. |
| **외인순매수(20d)** | 최근 20일 외국인 누적 순매수 금액 | - |
| **기관순매수(20d)** | 최근 20일 기관 누적 순매수 금액 | - |
| **스마트머니 승률** | 최근 20일 중 외인/기관 순매수 발생 일수 비율 | 꾸준히 매집하는지 확인 (60% 이상이면 강한 매집). |
| **양매수 비율** | 최근 20일 중 외인/기관 동시 순매수 일수 비율 | 강력한 시세 분출 전조 신호 중 하나. |
| **VCP 신호** | O/X (가격 변동성 축소 + 거래량 축소 + 스마트머니) | 주가가 수렴하며 에너지를 응축하다 상승하기 직전의 변동성 축소 패턴. |

## 9. TTM 실적 & 기본 재무 (TTM & Basic Financials)
가장 최근의 12개월 합산(Trailing Twelve Months) 및 최신 분기말 재무 수치입니다. **(모든 금액 지표의 기본 단위: 억원)**

| 지표명 | 상세 정의 (단위: 억원) |
| :--- | :--- |
| **TTM 매출 / TTM 영업이익 / TTM 순이익** | 최근 4분기 합산 실적 수치 |
| **TTM 영업CF / TTM CAPEX / TTM FCF** | 최근 4분기 합산 현금흐름 수치 |
| **자본 / 부채 / 자산총계** | 가장 최근 분기말 기준 장부상 상태 |

## 10. 연속 성장 & 이익률 추이 (Consecutive Growth & Margins)
일시적인 실적 상승이 아닌 '구조적인 성장'을 판별합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **연속성장** | 매출 / OP(영업이익) / NP(순이익) / CF(영업CF)의 연속 증가 년수 | 장기 추세적 성장을 증명하는 횟수. |
| **흑자전환** | 전년 적자 → 최신 흑자 전환 시 O | 강력한 주가 상승의 촉매. |
| **이익률 개선 / 이익률 급개선** | O/X (영업이익률 전년 대비 증가 / 5%p 이상 증가) | 제품의 판가 인상이나 비용 절감 성공을 의미. |
| **이익률 변동폭** | 최근 영업이익률 - 전년 영업이익률 (pp) | 마진 폭 확장 정도. |
| **실적 가속 연속** | 매출/영업이익의 YoY 증가폭(ΔYoY)이 2분기 연속 확대 여부 | 이익 성장 속도 자체가 가팔라지는 강력한 모멘텀 신호. |

---

## 11. 효율성 & 마진 (Efficiency & Quality Margin)
투여된 자본 대비 실질적 현금 및 마진 창출력을 평가합니다.

| 지표명 | 상세 정의 및 계산식 | 투자 해석 가이드 |
| :--- | :--- | :--- |
| **ROIC% / ROIC 전년%** | 세후영업이익 / 영업투하자본 (초과현금 차감) | ROE의 부채 왜곡을 제거한 진짜 영업 수익성. |
| **ROIC 개선** | 최신 ROIC > 전년 ROIC 시 O | - |
| **GPM 최근% / 전년%** | (매출총이익 / 매출액) × 100 | 원가 통제력. 워런 버핏은 이 수치가 높고 유지되는 기업을 선호. |
| **GPM 변화(pp)** | 최신 GPM - 전년 GPM | 양수(+)면 원가율이 개선되고 있음. |
| **퀄리티 턴어라운드** | O/X (GPM 2%p 상승 & 영업CF 흑자 & ROIC 개선) | 단순 적자 축소가 아닌, 원가/수익성/현금흐름이 3박자로 개선되는 찐 턴어라운드. |

## 12. 전략 점수 산정 로직 (Strategy Scores)
각 테마별 투자 전략에 부합하는 정도를 0~100점의 퍼센타일 랭크(백분위)로 수치화한 것입니다.

| 전략 점수명 | 반영되는 핵심 지표 및 가중치 |
| :--- | :--- |
| **종합점수** | 성장성(33.3%), 안정성(33.3%), 가격(33.3%) 세 축의 균등 밸런스 점수. |
| **성장성 점수** | OP CAGR(35) + 매출 CAGR(30) + Q_OP_YoY(25) + 실적가속(10) |
| **안정성 점수** | ROE(40) + F-스코어(35) + FCF수익률(25) |
| **가격 점수** | PER역순(40) + S-RIM괴리율(35) + PBR역순(25) |
| **주도주 점수** | RS등급(25) + 수급강도(20) + OP_CAGR(15) + Q_OP_YoY(15) + 거래대금(10) + 실적가속(10) + RSI(5) |
