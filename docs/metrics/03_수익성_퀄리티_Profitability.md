## 1. ROE (자기자본이익률)

### 정의
**Return on Equity** — 자기자본 대비 순이익의 비율로, 주주가 투자한 자본에 대한 수익률입니다. 워런 버핏이 가장 중시하는 지표 중 하나입니다.

### 산출 공식

```
ROE(%) = (TTM 순이익 / 자본총계) × 100
```

### 해석 기준

| ROE 범위 | 해석 |
|----------|------|
| < 0% | 적자 (자본 훼손 중) |
| 0% ~ 5% | 저수익 (자본비용 미달 가능) |
| 5% ~ 10% | 보통 |
| 10% ~ 15% | 양호 |
| 15% ~ 25% | 우수 (경쟁우위 보유) |
| > 25% | 탁월 (높은 레버리지 여부 점검) |

### 주의사항
- **높은 ROE + 높은 부채비율** = 레버리지에 의한 착시 가능
- DuPont 분해: `ROE = 순이익률 × 자산회전율 × 재무레버리지`
- S-RIM 모델에서 ROE > Ke(할인율)이면 초과이익 창출 기업

### 시스템 내 활용
- **종합점수 안정 축**: ROE 비중 40%
- **우량가치 Track B (금융주)**: ROE ≥ 8% 하드 필터
- **S-RIM 적정가치 산출**: 핵심 입력변수

## 2. ROIC (투하자본수익률)

### 정의
**Return on Invested Capital** — 실제 투자된 자본(부채+자기자본 중 영업에 투입된 부분) 대비 세후 영업이익의 비율입니다. 조엘 그린블라트(Joel Greenblatt)의 마법 공식에서 핵심 지표입니다.

### 산출 공식

```
NOPAT = 영업이익 × (1 - 0.22)        [실효세율 22%]
투하자본 = 총자산 - 유동부채 - 초과현금
초과현금 = max(0, 현금+단기금융상품 - 운영현금필요량)
운영현금필요량 = 매출액 × 2.5%

ROIC(%) = (NOPAT / 투하자본) × 100
```

### ROE와의 차이

| 비교 항목 | ROE | ROIC |
|----------|-----|------|
| 분자 | 순이익 | NOPAT (세후 영업이익) |
| 분모 | 자기자본 | 투하자본 (영업 투입 자본) |
| 레버리지 영향 | 있음 | 없음 (자본구조 중립) |
| 비교 적합성 | 같은 업종 | 업종 간 비교 가능 |
| 경제적 의미 | 주주 수익률 | 사업 수익률 |

### 해석 기준

| ROIC 범위 | 해석 |
|----------|------|
| < 5% | 자본비용 미달 (가치 파괴) |
| 5% ~ 10% | 보통 |
| 10% ~ 15% | 양호 (경쟁우위 시사) |
| 15% ~ 25% | 우수 (강한 경제적 해자) |
| > 25% | 탁월 (독점적 지위 가능) |

### 시스템 내 활용
- **우량가치 전략 Track A**: ROIC ≥ 10% 하드 필터 (그린블라트 해자 기준)
- **우량가치 전략 점수**: ROIC 비중 25%
- **현금배당 전략 점수**: ROIC Moat 비중 15%
- **돌파신호 펀더멘탈 품질**: ROIC 개선 ×20점
- **ROIC 개선 플래그**: 전년 대비 ROIC 증가 시 1 (퀄리티 턴어라운드 구성요소)

## 3. 영업이익률

### 정의
**Operating Margin** — 매출액 대비 영업이익의 비율로, 본업의 수익성을 나타냅니다.

### 산출 공식

```
영업이익률(%) = (영업이익 / 매출액) × 100
```

### 관련 파생 지표

| 지표 | 공식 | 설명 |
|------|------|------|
| **영업이익률 (최근)** | 최근 연도 영업이익률 | 현재 수익성 |
| **영업이익률 (전년)** | 이전 연도 영업이익률 | 비교 기준 |
| **마진 변화 (pp)** | 최근 - 전년 | 퍼센티지 포인트 변화 |
| **마진 개선 플래그** | 1 if 최근 > 전년, else 0 | 이진 개선 신호 |
| **마진 급개선** | 1 if 변화 ≥ 5pp, else 0 | 턴어라운드 신호 |

### 마진 급개선 기준

```
마진_급개선 = 1  if (영업이익률_최근 - 영업이익률_전년) ≥ 5%p
```

5%p 이상의 마진 개선은 구조적 변화(비용 절감, 제품 믹스 개선, 가격 인상력)를 시사합니다.

### 시스템 내 활용
- **턴어라운드 전략**: 마진 급개선 = 진입 조건 (하드 필터)
- **턴어라운드 전략 점수**: 순이익률 변화 비중 10%

## 4. 매출총이익률 (GPM)

### 정의
**Gross Profit Margin** — 매출액 대비 매출총이익(매출-매출원가)의 비율로, 제품/서비스의 기본 수익성을 나타냅니다.

### 산출 공식

```
GPM(%) = (매출총이익 / 매출액) × 100
GPM 변화(pp) = GPM_최근 - GPM_전년
```

### GPM vs 영업이익률 차이

| 비교 항목 | GPM | 영업이익률 |
|----------|-----|----------|
| 제외 비용 | 매출원가만 | 매출원가 + 판관비 |
| 의미 | 제품 자체 수익성 | 사업 운영 수익성 |
| 변동 요인 | 원재료비, 제조비 | GPM + 인건비, 마케팅비 |

### 시스템 내 활용
- **F-Score F8**: GPM 개선 여부
- **퀄리티 턴어라운드 신호**: GPM 변화 ≥ 2pp 필요
- **돌파신호 펀더멘탈 품질**: GPM 변화 > 0이면 최대 15점

## 5. 순이익률

### 정의
**Net Profit Margin** — 매출액 대비 순이익의 비율입니다.

### 산출 공식

```
순이익률(%) = (순이익 / 매출액) × 100
```

### 활용 포인트
- 영업이익률과 순이익률의 차이 = 금융비용, 법인세, 비영업 손익의 영향
- 큰 괴리 → 일회성 손익이나 이자비용 점검 필요

## 6. 자산회전율

### 정의
**Asset Turnover** — 총자산 대비 매출액의 비율로, 자산을 얼마나 효율적으로 활용하여 매출을 창출하는지 측정합니다.

### 산출 공식

```
자산회전율 = 매출액 / 총자산
```

### 해석
- 높을수록 자산 활용 효율이 높음
- 업종에 따라 정상 범위가 크게 다름 (소매업 높음, 자본집약 산업 낮음)

### 시스템 내 활용
- **F-Score F9**: 자산회전율 개선 여부

## 7. FCF & 현금흐름 지표

### 7.1 영업현금흐름 (Operating Cash Flow)

```
영업CF = 재무제표 현금흐름표의 영업활동현금흐름
```

순이익과 달리 회계적 조정을 제거한 **실제 현금 유입**을 나타냅니다.

### 7.2 CAPEX (자본적 지출)

```
CAPEX = |투자활동현금흐름 중 유형자산 취득|  (절대값)
```

### 7.3 FCF (잉여현금흐름)

```
FCF = 영업CF - CAPEX
```

투자에 필요한 자본을 제외한 후 **주주에게 돌아갈 수 있는 현금**입니다.

### 7.4 FCF Yield (잉여현금흐름 수익률)

```
FCF Yield(%) = (TTM FCF × 단위배수 / 시가총액) × 100
```

채권의 수익률과 유사한 개념으로, 주식을 현금흐름 자산으로 볼 때의 수익률입니다.

| FCF Yield | 해석 |
|-----------|------|
| < 0% | FCF 음수 (투자 과다 또는 영업CF 부족) |
| 0% ~ 3% | 보통 |
| 3% ~ 6% | 양호 |
| 6% ~ 10% | 매력적 |
| > 10% | 매우 매력적 (다만 성장 둔화 가능) |

### 7.5 현금전환율 (FCF Conversion Rate)

```
현금전환율(%) = (영업CF / 순이익) × 100
```

| 현금전환율 | 해석 |
|-----------|------|
| > 120% | 우수 (감가상각 등 비현금 비용이 큼 → 현금 풍부) |
| 80% ~ 120% | 양호 (이익이 현금으로 잘 전환) |
| 50% ~ 80% | 주의 (운전자본 증가, 매출채권 문제 가능) |
| < 50% | 경고 (이익의 질 의심, 분식 가능성) |

### 7.6 CAPEX 비율

```
CAPEX 비율(%) = (CAPEX / 영업CF) × 100
```

| CAPEX 비율 | 해석 |
|-----------|------|
| < 30% | 경자산 사업 (소프트웨어, 플랫폼) |
| 30% ~ 60% | 보통 |
| 60% ~ 80% | 자본집약적 (제조, 인프라) |
| > 80% | 과도한 투자 (FCF 거의 없음) |

### 시스템 내 활용
- **종합점수 안정 축**: FCF Yield 비중 25%
- **우량가치 전략 점수**: FCF Yield 비중 25%
- **현금배당 전략**: FCF Yield ≥ 3% 하드 필터, 점수 비중 25%
- **고성장 전략**: TTM 영업CF > 0 하드 필터

## 8. 이익의 질 (Earnings Quality)

### 정의
순이익이 실제 현금흐름에 의해 뒷받침되는지를 판단합니다.

### 이익의 질 플래그

```
이익의_질 = 1  if (영업CF > 순이익) AND (순이익 > 0)
이익의_질 = 0  otherwise
```

### 해석

| 상황 | 순이익 | 영업CF | 이익의 질 | 의미 |
|------|--------|--------|----------|------|
| 최상 | 양수 | 순이익보다 큼 | 높음 | 진짜 돈을 벌고 있음 |
| 양호 | 양수 | 양수 (순이익 이하) | 보통 | 운전자본 증가 모니터링 |
| 주의 | 양수 | 음수 | 낮음 | 이익은 있지만 현금 유출 |
| 위험 | 음수 | 음수 | - | 사업 부진 |

### 부채상환능력

```
부채상환능력 = 영업CF / 총부채
```

영업현금흐름으로 총부채를 갚을 수 있는 능력을 측정합니다.

## 9. 재무 안정성 지표

### 9.1 유동비율

```
유동비율(%) = (유동자산 / 유동부채) × 100
```

| 유동비율 | 해석 |
|---------|------|
| < 100% | 단기 지급능력 부족 (경고) |
| 100% ~ 150% | 보통 |
| 150% ~ 200% | 양호 |
| > 200% | 안전 (다만 자산 효율성 점검) |

### 9.2 부채비율

```
부채비율(%) = (총부채 / 자본총계) × 100
```

| 부채비율 | 해석 |
|---------|------|
| < 50% | 보수적 재무구조 |
| 50% ~ 100% | 양호 |
| 100% ~ 200% | 보통 (업종별 상이) |
| > 200% | 높은 레버리지 (리스크 관리 필요) |

### 9.3 이자보상배율

```
이자보상배율 = TTM 영업이익 / TTM 이자비용
이자비용 없는 경우: 99 (실질 무차입)
```

| 이자보상배율 | 해석 |
|------------|------|
| < 1.0 | 영업이익으로 이자도 못 갚음 (위험) |
| 1.0 ~ 3.0 | 취약 |
| 3.0 ~ 5.0 | 보통 |
| 5.0 ~ 10.0 | 양호 |
| > 10.0 | 안전 |

### 시스템 내 활용
- **우량가치 전략 Track A**: 부채비율 < 120%, 유동비율 > 120% 하드 필터
- **현금배당 전략**: 부채비율 < 120% 하드 필터, 부채비율 역수 점수 5%
- **턴어라운드 전략**: 이자보상배율 > 1.5 하드 필터 (파산 방지)

## 10. 턴어라운드 신호

### 10.1 턴어라운드 전환

```
턴어라운드 = 1  if (전년 순이익 < 0) AND (금년 순이익 > 0)
```

적자에서 흑자로의 전환을 감지합니다.

### 10.2 퀄리티 턴어라운드 (복합 신호)

```
퀄리티_턴어라운드 = 1  if ALL of:
  ├── GPM 변화 ≥ 2%p (매출총이익률 개선)
  ├── TTM 영업CF > 0 (현금흐름 양호)
  └── ROIC 개선 = 1 (자본효율 개선)
```

단순 턴어라운드와 달리, 수익성의 **질적 개선**을 모두 확인합니다.

### 시스템 내 활용
- **턴어라운드 전략**: 턴어라운드 플래그 OR 마진 급개선 = 진입 조건
- **턴어라운드 전략 점수**: 턴어라운드 플래그 15%, 퀄리티 턴어라운드 15%
- **돌파신호 펀더멘탈 품질**: 턴어라운드 ×15점

## 11. 실전 활용 가이드

### 11.1 수익성 분석 프레임워크

```
[수익성 분석 순서]
1단계: ROE로 전체 주주 수익률 확인
  └── ROE > 15% → 우수

2단계: ROIC로 레버리지 제거한 순수 사업 수익률 확인
  └── ROE 높은데 ROIC 낮음 → 레버리지에 의한 ROE
  └── ROIC > 10% → 경제적 해자(moat) 보유

3단계: 마진 추이로 수익성 방향 확인
  └── GPM 개선 → 제품 경쟁력 향상
  └── 영업이익률 개선 → 비용 효율화 또는 가격 인상력

4단계: 현금흐름으로 이익의 질 검증
  └── 영업CF > 순이익 → 이익의 질 양호
  └── FCF Yield > 5% → 현금흐름 매력적
```

### 11.2 수익성 × 밸류에이션 매트릭스

| | 저PER/PBR | 고PER/PBR |
|---|----------|----------|
| **고ROE/ROIC** | **최고 매수 후보** (저평가 우량주) | 적정가 또는 성장 프리미엄 |
| **저ROE/ROIC** | 밸류 트랩 위험 | **회피** (고평가 저품질) |

### 11.3 업종별 핵심 수익성 지표

| 업종 | 핵심 지표 | 이유 |
|------|----------|------|
| 제조업 | ROIC, GPM | 자본 효율과 원가 경쟁력 |
| IT/SaaS | 영업이익률, FCF | 규모의 경제와 현금 창출 |
| 금융 | ROE, 부채비율 | 레버리지 사업 특성 |
| 유통/소매 | 자산회전율, 영업이익률 | 박리다매 vs 고마진 |
| 바이오 | 현금보유, FCF | 흑자전환 전까지 생존 능력 |

### 11.4 이익의 질 레드플래그

| 경고 신호 | 상세 |
|----------|------|
| 영업CF < 순이익 (3년 연속) | 매출채권 미회수, 재고 쌓임 가능 |
| 현금전환율 < 50% | 이익이 현금으로 전환 안 됨 |
| FCF 지속 음수 | 과도한 투자 또는 사업 부진 |
| 부채비율 급증 + ROE 유지 | 레버리지로 ROE 유지하는 위험 신호 |
| 이자보상배율 < 1.5 | 이자 부담이 영업이익 초과 임박 |
