## 1. DPS (주당배당금)

### 정의
**Dividend Per Share** — 보통주 1주당 지급되는 연간 현금 배당금입니다.

### 데이터 소스
- FnGuide HIGHLIGHT 테이블의 **연간 배당금** (배당금(보통주,현금))
- **분기배당, 추정치는 제외** — 확정된 연간 배당만 사용

### 주의사항
- 중간배당을 실시하는 기업의 경우 연간 합계가 반영됨
- 주식배당, 현물배당은 제외 (현금배당만)
- 최근 1~2년 내 배당 정책 변경 가능성 항상 점검

## 2. 배당수익률

### 정의
현재 주가 대비 연간 배당금의 비율로, 배당만으로 얻을 수 있는 투자 수익률입니다.

### 산출 공식

```
배당수익률(%) = (DPS_최근 / 현재 주가) × 100
```

### 해석 기준

| 배당수익률 | 해석 |
|-----------|------|
| 0% | 무배당 |
| 0.1% ~ 1.0% | 상징적 배당 |
| 1.0% ~ 2.0% | 성장주 수준 |
| 2.0% ~ 4.0% | 적정 배당 |
| 4.0% ~ 6.0% | 고배당 |
| 6.0% ~ 10.0% | 초고배당 (지속성 점검) |
| > 10.0% | 비정상적 고배당 (주가 하락 또는 특별배당 가능) |

### 시스템 내 활용
- **우량가치 Track B (금융주)**: 배당수익률 ≥ 2% 하드 필터
- **현금배당 전략**: 배당수익률 ≥ 1% 하드 필터, 점수 비중 20%

## 3. 배당성향 (Payout Ratio)

### 정의
순이익 중 배당금으로 지급하는 비율입니다. 배당의 **지속가능성**을 판단하는 핵심 지표입니다.

### 산출 공식

```
배당성향(%) = (DPS / EPS) × 100
```

### 해석 기준

| 배당성향 | 해석 |
|---------|------|
| < 20% | 배당 여력 충분, 성장 재투자 우선 |
| 20% ~ 40% | 균형잡힌 배당 정책 |
| 40% ~ 60% | 적극적 배당 (성숙 기업) |
| 60% ~ 80% | 높은 배당, 이익 변동 시 유지 곤란 가능 |
| > 80% | **경고**: 이익 대부분을 배당으로 지급, 감소 리스크 |
| > 100% | **위험**: 이익보다 많은 배당 (적립금 소진) |

### 시스템 내 활용
- **현금배당 전략**: 배당성향 < 80% 하드 필터 (지속성 확보)
- **현금배당 전략 점수**: 배당성향 역수(낮을수록 좋음) 비중 10%
- **밸류 트랩 경고**: 배당성향 > 80% 시 발동

---

## 4. DPS CAGR (배당성장률)

### 정의
배당금의 복합연평균성장률입니다. 배당 투자에서 **배당 성장**은 단순 높은 배당수익률보다 장기적으로 더 중요합니다.

### 산출 공식

```
DPS CAGR = (최근 DPS / 과거 DPS)^(1/연수) - 1
```

### 산출 조건
- 최소 2개년 배당 데이터 필요
- 초기와 최종 DPS 모두 양수 필요

### 해석 기준

| DPS CAGR | 해석 |
|----------|------|
| < 0% | 배당 감소 추세 |
| 0% ~ 5% | 인플레이션 수준 증가 |
| 5% ~ 10% | 안정적 배당 성장 |
| 10% ~ 20% | 적극적 배당 확대 |
| > 20% | 매우 높은 배당 성장 (저기저 효과 점검) |

### 시스템 내 활용
- **현금배당 전략 점수**: DPS CAGR 비중 15%

## 5. 배당 연속 성장

### 정의
최근 시점부터 역순으로 매년 배당금이 증가한 연속 횟수입니다. **배당 귀족주(Dividend Aristocrats)** 선별의 핵심 기준입니다.

### 산출 로직

```python
연속 = 0
for i in range(최근, 과거):
    if DPS[i] > DPS[i-1]:
        연속 += 1
    else:
        break
```

### 배당 연속 성장 보너스

시스템에서는 연속 성장 연수에 따라 **로그 스케일 보너스**를 부여합니다:

```
보너스 = min(log₂(연속연수 + 1) × 3, 10)
```

| 연속 연수 | 보너스 점수 | 의미 |
|----------|-----------|------|
| 1년 | +2.1점 | 최근 1회 증가 |
| 2년 | +3.4점 | 2년 연속 증가 |
| 3년 | +4.5점 | 배당 성장 추세 형성 |
| 5년 | +5.6점 | 안정적 배당 성장 기업 |
| 7년 | +7.2점 | 배당 성장 우수 |
| 10년 | +7.5점 | 배당 귀족 후보 |
| 15년+ | +10.0점 | 최대 (캡) |

### 로그 스케일 적용 이유
- 초기 1~3년 배당 성장 시작은 큰 의미가 있으므로 급격히 보상
- 10년 이상은 이미 충분히 검증되었으므로 한계체감 적용
- 최대 10점 캡으로 과도한 보너스 방지

## 6. 배당+이익 동반성장 보너스

### 정의
배당 성장이 이익 성장에 의해 **뒷받침**되는 경우 추가 보너스를 부여합니다.

### 발동 조건

```
배당_이익_동반성장 = True  if:
  ├── 이익 연속 성장 ≥ 2년
  └── 배당 연속 성장 ≥ 1년

보너스: +2점 (현금배당 전략 점수에 가산)
```

### 해석
이익이 뒷받침되는 배당 성장은 지속가능성이 높습니다. 반대로 이익이 정체/감소하면서 배당만 증가하는 것은 위험 신호입니다.

## 7. 밸류 트랩 경고

### 정의
**Value Trap Warning** — 높은 배당수익률이 함정(트랩)이 될 수 있는 경우를 감지합니다. 겉보기에 매력적인 배당이 실제로는 지속 불가능하거나 주가 하락의 결과인 경우입니다.

### 발동 조건 (OR 조건, 하나라도 해당 시 경고)

```
밸류트랩경고 = 1  if ANY of:
  ├── 배당성향 > 80%
  │     └── 이익 대부분을 배당으로 지급 → 감소 가능
  │
  ├── (배당수익률 > 10%) AND (RS등급 < 30)
  │     └── 초고배당 + 주가 하락 추세 → 배당이 아니라 주가 하락 때문
  │
  └── 현금전환율 < 70%
        └── 이익이 현금으로 전환되지 않음 → 배당 재원 부족
```

### 밸류 트랩 페널티

```
경고 발동 시: 배당 전략 점수 × 0.7 (30% 감점)
```

### 밸류 트랩의 전형적 패턴

| 패턴 | 설명 | 결과 |
|------|------|------|
| 주가 하락형 | 주가 급락으로 배당수익률 급상승 | 향후 배당 감소 + 추가 주가 하락 |
| 이익 감소형 | 과거 이익 기준 배당, 현재 이익 감소 | 배당 삭감 불가피 |
| 현금 부족형 | 회계상 이익은 있지만 현금 부족 | 차입 배당 또는 배당 중단 |
| 일회성 배당 | 자산매각 등으로 특별배당 | 반복 불가, 수익률 왜곡 |

## 8. 실전 활용 가이드

### 8.1 배당투자 분석 프레임워크

```
[배당주 분석 순서]
1단계: 배당수익률 확인 (최소 2% 이상)
  └── 과거 3년간 배당수익률 추이도 확인

2단계: 배당 지속가능성 점검
  ├── 배당성향 < 60% (여유 있음)
  ├── FCF > 배당 총액 (현금으로 감당 가능)
  └── 현금전환율 > 80% (이익의 질 양호)

3단계: 배당 성장성 확인
  ├── DPS CAGR > 5% (배당 꾸준히 증가)
  ├── 배당 연속 성장 ≥ 3년
  └── 이익 동반 성장 여부

4단계: 밸류 트랩 점검
  ├── RS등급 확인 (주가 하락 추세 아닌지)
  ├── 이익 감속 경고 확인
  └── 밸류 트랩 경고 플래그 확인

5단계: 전체 재무건전성 확인
  ├── F-Score ≥ 5
  ├── 부채비율 < 120%
  └── 이자보상배율 > 3
```

### 8.2 현금배당 전략 (시스템 내 구현)

#### 하드 필터

```
✓ FCF Yield ≥ 3%
✓ 배당수익률 ≥ 1%
✓ 부채비율 < 120%
✓ 시가총액 ≥ 500억원
✓ 배당성향 < 80%
✓ 현금전환율 ≥ 70%
```

#### 스코어링

```
기본 점수:
├── FCF Yield:          25%
├── 배당수익률:          20%
├── DPS CAGR:           15%
├── ROIC Moat:          15%
├── 배당성향 역수:       10%
├── F-Score:            10%
└── 부채비율 역수:        5%

보너스 가산:
├── 배당 연속 성장 보너스: +0 ~ +10점
└── 배당+이익 동반성장:    +2점

페널티:
└── 밸류 트랩 경고 시:    ×0.7 (30% 감점)

과열 감도: 0.30 (5개 전략 중 두 번째로 높음)
```

### 8.3 배당 투자 스타일별 접근

#### 스타일 A: 고배당 수익형

```
목표: 현재 높은 배당수익률
중점 지표: 배당수익률, FCF Yield
리스크: 밸류 트랩
보완 조건: 배당성향 < 60%, F-Score ≥ 6
```

#### 스타일 B: 배당 성장형

```
목표: 장기 배당금 증가
중점 지표: DPS CAGR, 배당 연속 성장, 이익 CAGR
리스크: 현재 수익률 낮을 수 있음
보완 조건: 이익 동반 성장, 배당성향 여유
```

#### 스타일 C: 배당+가치 복합형

```
목표: 저평가 고배당
중점 지표: 배당수익률 + PBR + S-RIM 괴리율
리스크: 저평가에는 이유가 있을 수 있음
보완 조건: F-Score ≥ 7, ROE > 10%
```

### 8.4 배당 투자 시 흔한 실수

| 실수 | 설명 | 방지 방법 |
|------|------|----------|
| 수익률만 보기 | 높은 수익률 = 좋은 배당주? | 배당성향, FCF, RS등급 함께 확인 |
| 과거 배당 신뢰 | "지금까지 잘 줬으니까" | 이익 추세, 현금흐름 점검 |
| 세금 무시 | 배당소득세 15.4% + 금융소득 종합과세 | 세후 실질 수익률로 판단 |
| 배당락 무시 | 배당 전 매수 → 배당락 하락 | 배당락 이후 매수 고려 |
| 분산 미흡 | 고배당 소수 종목 집중 | 업종/섹터 분산 |

### 8.5 과열 감도와 배당

```
배당 전략 과열 감도: 0.30

의미: 과열점수가 40 이상일 때 배당 전략 점수에 최대 30% 감점
이유: 금리 상승 → 배당주 매력 감소 (채권 대비 열위)
      주가 급등 → 배당수익률 하락

계산: 최종점수 = 배당점수 × (1 - clip((과열-40)/60, 0, 1) × 0.30)
```
