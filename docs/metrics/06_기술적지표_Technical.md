## 1. 이동평균선 (MA)

### 정의
**Moving Average** — 일정 기간의 종가 평균으로, 가격 추세를 부드럽게 보여줍니다.

### 산출 공식

```
MA(N) = (최근 N일간 종가의 합) / N
```

### 적용 이동평균

| 이동평균 | 기간 | 필요 데이터 | 의미 |
|---------|------|-----------|------|
| **MA20** | 20일 | ≥ 20 거래일 | 단기 추세 (약 1개월) |
| **MA60** | 60일 | ≥ 60 거래일 | 중기 추세 (약 3개월) |

### 주가 vs 이동평균 괴리율

```
MA20 괴리율(%) = ((종가 / MA20) - 1) × 100
MA60 괴리율(%) = ((종가 / MA60) - 1) × 100
```

| 괴리율 | 해석 |
|--------|------|
| > +10% | 단기 과열 (평균 대비 크게 상승) |
| +3% ~ +10% | 강한 상승 추세 |
| -3% ~ +3% | 이동평균 근접 (지지/저항 테스트) |
| -10% ~ -3% | 약세 추세 |
| < -10% | 과매도 영역 |

### 시스템 내 활용
- **과열점수**: MA20 괴리율 비중 25% (괴리율 +20% → 100점)
- **돌파신호 가격패턴**: MA20 근접도 (0~10% 범위에서 3%가 이상적)

## 2. 52주 고가/저가

### 정의
최근 52주(약 1년) 동안의 최고가와 최저가입니다. 중장기 가격 범위를 정의합니다.

### 산출 공식

```
52주 고가 = max(최근 52주간 종가)
52주 저가 = min(최근 52주간 종가)

고가 근접도(%) = ((종가 / 52주 고가) - 1) × 100
저가 이격도(%) = ((종가 / 52주 저가) - 1) × 100
```

### 해석 기준

#### 52주 고가 근접도

| 근접도 | 해석 |
|--------|------|
| 0% (= 52주 신고가) | 강력한 상승 모멘텀 |
| -5% ~ 0% | 고가 근접, 돌파 시도 중 |
| -10% ~ -5% | 고가 대비 소폭 조정 |
| -20% ~ -10% | 조정 국면 |
| < -30% | 약세 국면 또는 하락 추세 |

#### 52주 저가 이격도

| 이격도 | 해석 |
|--------|------|
| > +100% | 저점 대비 2배 이상 상승 |
| +50% ~ +100% | 강한 상승 추세 |
| +20% ~ +50% | 양호한 상승 |
| 0% ~ +20% | 저가 근접, 바닥권 |

### 시스템 내 활용
- **과열점수**: 52주 근접도 비중 30% (고가 근접 시 과열 증가)

## 3. RSI (상대강도지수)

### 정의
**Relative Strength Index** — J. Welles Wilder가 개발한 모멘텀 오실레이터로, 최근 가격 변동의 상승 강도를 0~100 사이로 나타냅니다.

### 산출 공식

```
Delta = 일별 종가 변동
Gain = 14일간 양(+) 변동의 평균
Loss = 14일간 음(-) 변동의 절대값 평균
RS = Gain / Loss
RSI = 100 - (100 / (1 + RS))

* Loss = 0인 경우: RSI = 100
* 필요 데이터: ≥ 15 거래일
```

### 해석 기준

| RSI 범위 | 해석 |
|----------|------|
| > 80 | **극과매수** — 과열 영역, 단기 조정 가능 |
| 70 ~ 80 | 과매수 — 추세 강하나 주의 필요 |
| 50 ~ 70 | 상승 추세 — 건강한 범위 |
| 40 ~ 50 | 중립/약세 전환 가능 |
| 30 ~ 40 | 과매도 — 반등 기대 가능 |
| < 30 | **극과매도** — 반등 기회, 단 하락 추세 지속 가능 |

### RSI의 한계
- 강한 추세에서는 RSI가 장기간 과매수/과매도 영역에 머무를 수 있음
- 단독 사용보다 가격 패턴, 거래량과 함께 사용 권장

### 시스템 내 활용
- **과열점수**: RSI 비중 25% (RSI ≥ 80 → 100점, ≤ 50 → 0점)
- **주도주 전략 점수**: RSI_14 비중 5%
- **돌파신호 가격패턴**: RSI 50~60 → 30점 (이상적 매수 영역)

## 4. 변동성 (Volatility)

### 정의
주가의 변동 정도를 측정합니다. 높은 변동성 = 높은 리스크이자 높은 기회입니다.

### 산출 공식

```
60일 변동성(%) = 일별 수익률의 표준편차 × √252 × 100
                (연율화, 60 거래일 기준)
```

| 구성요소 | 설명 |
|---------|------|
| 일별 수익률 | (당일 종가 / 전일 종가) - 1 |
| 표준편차 | 60일간 일별 수익률의 std |
| √252 | 연간 거래일 기준 연율화 |

### 해석 기준

| 연율화 변동성 | 해석 |
|-------------|------|
| < 15% | 저변동 (대형 우량주) |
| 15% ~ 25% | 보통 (시장 평균) |
| 25% ~ 40% | 높음 (중소형주 또는 변동기) |
| 40% ~ 60% | 매우 높음 (테마주, 이벤트 주도) |
| > 60% | 극단적 (투기적 종목 또는 위기 상황) |

### 활용 포인트
- 변동성이 줄어드는 구간 = **변동성 수축** → VCP 패턴의 핵심 조건
- 변동성 확대 = 방향성 결정 임박 (상하 모두 가능)

## 5. 거래량/거래대금 분석

### 5.1 평균 거래대금

```
20일 평균 거래대금 = 최근 20일간 (종가 × 거래량)의 평균
                    또는 거래대금 컬럼 직접 사용

5일 평균 거래대금 = 최근 5일간 (종가 × 거래량)의 평균
```

| 20일 평균 거래대금 | 유동성 수준 |
|------------------|-----------|
| < 1억원 | 유동성 매우 부족 (매매 어려움) |
| 1억 ~ 5억원 | 소형주 수준 |
| 5억 ~ 50억원 | 중형주 수준 |
| 50억 ~ 500억원 | 대형주 수준 |
| > 500억원 | 초대형 (시장 주도주) |

### 5.2 거래대금 변화율

```
거래대금 변화율(%) = ((5일 평균 / 20일 평균) - 1) × 100
```

| 변화율 | 해석 |
|--------|------|
| > +100% | 거래 폭발적 증가 (주요 이벤트) |
| +30% ~ +100% | 거래 활성화 (관심도 증가) |
| -30% ~ +30% | 보통 범위 |
| < -30% | 거래 감소 (관심 이탈 또는 조정 안정) |

### 5.3 가격-거래량 다이버전스 (VPD)

```
가격-거래량 다이버전스 = min(|20일 수익률| + |거래대금 변화율|, 100)
조건: 주가 상승 + 거래량 감소 시에만 계산
```

**주가가 오르는데 거래량이 줄어드는 것**은 상승의 에너지가 약해지고 있다는 약세 신호입니다.

### 시스템 내 활용
- **주도주 전략**: 20일 평균 거래대금 비중 10%, 하드 필터 > 5억원
- **과열점수**: 가격-거래량 다이버전스 비중 20%
- **돌파신호 수급 유입**: 거래대금 변화율 > 0 시 최대 20점

## 6. RS등급 (상대강도)

### 정의
**Relative Strength Grade** — 윌리엄 오닐(William O'Neil)의 투자 방법론에서 핵심 지표로, 개별 종목의 수익률을 시장 전체(KOSPI/KOSDAQ) 대비 상대적으로 평가한 등급입니다.

### 산출 과정 (3단계)

#### 1단계: 기간별 원시 RS 산출

```
RS_60일  = 종목 60일 수익률(%) - 시장지수 60일 수익률(%)
RS_120일 = 종목 120일 수익률(%) - 시장지수 120일 수익률(%)
RS_250일 = 종목 250일 수익률(%) - 시장지수 250일 수익률(%)
```

- 시장지수: KOSPI(KS11) 또는 KOSDAQ(KQ11) — 종목의 시장구분에 따라 선택
- 데이터 소스: FinanceDataReader → 실패 시 yfinance 폴백

#### 2단계: 퍼센타일 랭킹

각 기간별 원시 RS를 전체 종목 대비 퍼센타일로 변환합니다:

```
Rank_60일  = percentile_rank(RS_60일)   → 0~100
Rank_120일 = percentile_rank(RS_120일)  → 0~100
Rank_250일 = percentile_rank(RS_250일)  → 0~100
```

#### 3단계: 가중 복합 RS

```
복합RS = (Rank_60일 × 0.4) + (Rank_120일 × 0.3) + (Rank_250일 × 0.3)
RS등급 = percentile_rank(복합RS)  → 최종 0~100
```

### 가중치 설계 이유

| 기간 | 가중치 | 이유 |
|------|--------|------|
| 60일 (약 3개월) | 40% | 최근 모멘텀에 가장 높은 비중 |
| 120일 (약 6개월) | 30% | 중기 추세 반영 |
| 250일 (약 1년) | 30% | 장기 추세 반영 |

### 해석 기준

| RS등급 | 해석 |
|--------|------|
| 90~100 | **최상위** — 시장 상위 10% 모멘텀 (주도주 영역) |
| 80~89 | **강세** — 시장 대비 매우 강한 상승 |
| 70~79 | **양호** — 시장 평균 이상 |
| 50~69 | **보통** — 시장과 유사한 움직임 |
| 30~49 | **약세** — 시장 대비 부진 |
| 0~29 | **최하위** — 시장 하위 30%, 하락 추세 |

### 시스템 내 활용
- **주도주 전략**: RS등급 비중 25%, RS등급 ≥ 80 하드 필터
- **고성장 전략**: RS등급 비중 25%, RS등급 ≥ 50 하드 필터
- **밸류 트랩 경고**: RS등급 < 30 + 배당수익률 > 10% 시 발동

## 7. 실전 활용 가이드

### 7.1 기술적 지표 통합 분석 프레임워크

```
[기술적 분석 순서]

1단계: 추세 확인 (방향)
  ├── 주가 > MA20 > MA60 → 상승 추세
  ├── MA20 > 주가 > MA60 → 조정 국면
  └── 주가 < MA20 < MA60 → 하락 추세

2단계: 모멘텀 확인 (강도)
  ├── RS등급 ≥ 80 → 시장 주도주
  ├── RSI 50~70 → 건강한 상승 모멘텀
  └── RSI > 80 → 과열 주의

3단계: 변동성 확인 (리스크)
  ├── 변동성 수축 중 → 돌파 준비 가능
  └── 변동성 확대 중 → 방향성 결정 구간

4단계: 거래량 확인 (참여도)
  ├── 주가↑ + 거래량↑ → 건강한 상승
  ├── 주가↑ + 거래량↓ → 상승 약화 (VPD)
  └── 주가↓ + 거래량↓ → 세력 이탈 없이 조정 중
```

### 7.2 매수/매도 타이밍 시그널

#### 매수 우호 조건

```
✓ RS등급 ≥ 70 (시장 대비 강세)
✓ RSI 40~60 (과매수 아닌 상태에서 진입)
✓ MA20 괴리율 0~5% (이동평균 근처에서 진입)
✓ 거래대금 증가 추세
✓ 변동성 수축 후 확대 시작
```

#### 매도/주의 조건

```
⚠ RSI > 80 (과매수)
⚠ MA20 괴리율 > 15% (단기 과열)
⚠ 52주 고가 근접 + 거래량 감소 (VPD 발생)
⚠ RS등급 급락 (모멘텀 붕괴)
```

### 7.3 RS등급을 활용한 모멘텀 투자

윌리엄 오닐의 CAN SLIM에서 RS등급은 가장 중요한 지표 중 하나입니다:

```
[RS 기반 모멘텀 전략]

매수 조건:
├── RS등급 ≥ 80 (상위 20%)
├── 최근 분기 이익 가속 (이익가속 = 1)
├── 거래대금 > 5억원/일
└── 수급강도 > 0 (기관/외인 매수)

보유 조건:
├── RS등급 ≥ 60 유지
└── MA20 위에서 거래 유지

매도 조건:
├── RS등급 < 50 하락 (모멘텀 상실)
├── MA60 하향 이탈
└── 과열점수 > 70 + 이익감속 경고
```

### 7.4 기술적 지표와 펀더멘탈 조합

| 기술적 신호 | 펀더멘탈 조건 | 해석 |
|------------|-------------|------|
| RS등급 ↑ + RSI 50~70 | 이익 가속 + ROE 개선 | **최적 매수** — 모멘텀+실적 동시 개선 |
| RS등급 ↑ + RSI > 80 | 이익 감속 경고 | **매도 검토** — 기술적 과열 + 펀더멘탈 악화 |
| RS등급 ↓ + RSI < 30 | F-Score ≥ 7 + ROIC > 10% | **역발상 매수 후보** — 일시적 조정 |
| RS등급 ↓ + RSI < 30 | 이익 감소 + FCF < 0 | **회피** — 하락에 이유 있음 |

### 7.5 주요 기술적 지표 요약표

| 지표 | 산출 기간 | 범위 | 해석 방향 |
|------|----------|------|----------|
| MA20 괴리율 | 20일 | -∞ ~ +∞ (%) | 0% 근접이 이상적 진입점 |
| MA60 괴리율 | 60일 | -∞ ~ +∞ (%) | 추세 확인용 |
| 52주 고가 근접도 | 52주 | -100% ~ 0% | 0%에 가까울수록 강세 |
| RSI | 14일 | 0 ~ 100 | 40~60 = 이상적 진입 |
| 60일 변동성 | 60일 | 0% ~ ∞ | 수축→확대가 돌파 신호 |
| RS등급 | 60/120/250일 | 0 ~ 100 | ≥80 = 주도주 |
| 거래대금 변화율 | 5일/20일 | -∞ ~ +∞ (%) | 증가 = 관심/모멘텀 |
