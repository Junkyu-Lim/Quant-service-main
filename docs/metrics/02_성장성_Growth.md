## 1. CAGR (복합연평균성장률)

### 정의
**Compound Annual Growth Rate** — 일정 기간 동안의 연평균 복합 성장률입니다. 단순 평균이 아닌 복리 개념의 성장률로, 성장의 일관성을 측정합니다.

### 산출 공식

```
CAGR = (최종값 / 초기값)^(1/연수) - 1
```

### 적용 대상

| CAGR 지표 | 데이터 소스 | 상한 캡 | 비고 |
|-----------|-----------|---------|------|
| **매출액 CAGR** | 연간 매출액 | 150% | 구조적 성장 측정 |
| **영업이익 CAGR** | 연간 영업이익 | 150% | 본업 성장 측정 |
| **순이익 CAGR** | 연간 순이익 | 150% | 최종 이익 성장 측정 |
| **영업CF CAGR** | 연간 영업현금흐름 | 상한 없음 | 현금 창출 성장 |
| **FCF CAGR** | 연간 잉여현금흐름 | 상한 없음 | 투자 후 잉여현금 성장 |
| **DPS CAGR** | 연간 배당금 | 상한 없음 | 배당 성장 측정 |

### 산출 조건
- **최소 2개년** 데이터 필요
- 초기값과 최종값 모두 **양수**여야 산출 가능
- 음수→양수 전환 시 CAGR 산출 불가 (턴어라운드 지표로 대체)

### CAGR 캡(상한) 설정 이유

```
CAGR_CAP = 150%  (매출, 영업이익, 순이익에 적용)
```

| 상황 | 예시 | 캡 적용 효과 |
|------|------|-------------|
| 소형주 초기 급성장 | 매출 10억→100억 (2년, CAGR 216%) | 150%로 제한 |
| 저기저 효과 | 순이익 1억→50억 (3년, CAGR 368%) | 150%로 제한 |
| 정상 고성장 | 매출 1000억→3000억 (3년, CAGR 44%) | 캡 미적용 |

### 해석 기준

| CAGR 범위 | 해석 |
|----------|------|
| < 0% | 역성장 (구조적 문제 점검 필요) |
| 0% ~ 5% | 저성장 (인플레이션 수준) |
| 5% ~ 15% | 안정 성장 |
| 15% ~ 30% | 고성장 |
| 30% ~ 50% | 매우 높은 성장 |
| > 50% | 초고성장 (지속성 점검 필요) |

## 2. 분기 YoY 성장률

### 정의
**Quarter-over-Quarter Year-on-Year Growth** — 최근 분기 실적을 전년 동기와 비교한 성장률입니다. 계절성을 제거한 가장 시의성 높은 성장 지표입니다.

### 산출 공식

```
분기 YoY(%) = ((당분기 실적 / 전년 동분기 실적) - 1) × 100
```

### 적용 대상

| YoY 지표 | 설명 | 상한 캡 |
|----------|------|---------|
| **분기 매출 YoY(%)** | 매출액 전년 동기 대비 | 300% |
| **분기 영업이익 YoY(%)** | 영업이익 전년 동기 대비 | 300% |
| **분기 순이익 YoY(%)** | 순이익 전년 동기 대비 | 300% |

### YoY 캡 설정

```
YoY_CAP = 300%
```

분기 실적은 기저효과가 더 크게 작용하므로 CAGR보다 높은 300% 상한을 적용합니다.

### 주의사항
- 전년 동기 실적이 0 또는 음수이면 산출 불가
- 일회성 요인(자산매각, 소송비용 등)에 의한 왜곡 가능
- **반드시 매출 YoY와 이익 YoY를 함께 확인** (매출 없이 이익만 성장 = 일회성 가능)

## 3. TTM YoY 성장률

### 정의
**Trailing Twelve Months YoY Growth** — 최근 4개 분기 합산(TTM)을 전년 동기 TTM과 비교한 성장률입니다. 분기 변동성을 완화하면서도 시의성을 유지합니다.

### 산출 공식

```
TTM = 최근 4분기 실적 합산
전년 TTM = 전년 동기 4분기 실적 합산
TTM YoY(%) = ((TTM / 전년 TTM) - 1) × 100
```

### 분기 YoY와의 차이

| 비교 항목 | 분기 YoY | TTM YoY |
|----------|---------|---------|
| 시의성 | 매우 높음 | 높음 |
| 변동성 | 높음 (단일 분기) | 낮음 (4분기 평균) |
| 계절성 제거 | 전년 동기 비교로 제거 | 전년 동기 비교로 제거 |
| 일회성 왜곡 | 영향 큼 | 영향 상대적 작음 |
| 적합 용도 | 모멘텀 포착 | 추세 확인 |

## 4. 연속 성장 카운터

### 정의
최근 시점부터 역순으로 매년 실적이 증가한 연속 횟수를 카운트합니다. 성장의 **일관성과 지속성**을 측정합니다.

### 산출 로직

```python
# 최근 연도부터 역순으로 검사
연속 = 0
for i in range(최근, 과거):
    if 실적[i] > 실적[i-1]:
        연속 += 1
    else:
        break  # 감소 발견 시 즉시 중단
```

### 적용 대상

| 카운터 | 대상 | 활용 |
|--------|------|------|
| **매출 연속 성장** | 연간 매출액 | 성장 지속성 |
| **영업이익 연속 성장** | 연간 영업이익 | 본업 성장 지속성 |
| **순이익 연속 성장** | 연간 순이익 | 이익 성장 지속성 |
| **영업CF 연속 성장** | 연간 영업현금흐름 | 현금 창출 안정성 |

### 해석 기준

| 연속 연수 | 의미 |
|----------|------|
| 0 | 직전년 대비 감소 (모멘텀 부재) |
| 1~2 | 단기 회복/성장 |
| 3~4 | 중기 성장 추세 |
| 5+ | 구조적 성장 (강력한 경쟁우위 시사) |

## 5. 이익 가속 신호

### 정의
**Earnings Acceleration** — 이익 성장률이 가속(증가)하고 있는지를 판단하는 이진 신호입니다. 윌리엄 오닐(William O'Neil)의 CAN SLIM 전략에서 핵심적인 개념입니다.

### 산출 로직

```
이익가속 = 1 (True) if:
    최근 2개 연속 분기의 YoY 변화폭(ΔYoY)이 양수
    즉, YoY 성장률이 2분기 연속 증가 중

이익가속 = 0 (False) otherwise
```

### 예시

| 분기 | 영업이익 YoY | ΔYoY | 가속? |
|------|-------------|------|------|
| Q1 | +10% | - | - |
| Q2 | +15% | +5%p | O |
| Q3 | +25% | +10%p | O → **가속 확인!** |
| Q4 | +20% | -5%p | X → 가속 해제 |

### 시스템 내 활용
- **주도주 전략**: 이익가속 비중 10%
- **고성장 전략**: 이익가속 비중 20%
- **종합점수 성장 축**: 이익가속 비중 10%
- **돌파신호 펀더멘탈 품질**: 이익가속 ×30점

## 6. 이익 감속 경고

### 정의
**Earnings Deceleration Warning** — 이익 모멘텀이 둔화되거나 악화되고 있음을 경고하는 신호입니다. 밸류에이션 트랩 방지에 핵심적입니다.

### 감속 경고 발동 조건 (3가지 유형)

| 유형 | 조건 | 해석 |
|------|------|------|
| **피크 이익** | 이전 YoY > 0 → 현재 YoY < 이전 YoY (감속) | 성장 정점 통과 |
| **적자 심화** | 순이익이 지속적으로 악화 | 구조적 문제 |
| **흑자→적자 전환** | 이전 양수 → 현재 음수 | 수익성 붕괴 |

### 시스템 내 영향
- **과열점수**: +10점 보너스 (경고 시)
- **종합점수 타이밍 축**: -15점 페널티

```
타이밍_기본 = 비과열(40%) + 돌파신호(45%)
타이밍_최종 = clip((타이밍_기본 + 감속페널티) / 0.85, 0, 100)
감속페널티 = -15 if 이익감속경고 = 1
```

## 7. 실전 활용 가이드

### 7.1 성장 지표 체계적 분석 프레임워크

```
[성장 분석 순서]
1단계: CAGR로 중장기 성장 추세 확인 (3~5년)
  └── 매출 CAGR > 영업이익 CAGR → 마진 하락 주의
  └── 영업이익 CAGR > 매출 CAGR → 레버리지 효과 (긍정)

2단계: 분기 YoY로 최근 모멘텀 확인
  └── CAGR 양호 + 분기 YoY 양호 → 성장 지속 (Best)
  └── CAGR 양호 + 분기 YoY 악화 → 성장 둔화 주의
  └── CAGR 부진 + 분기 YoY 양호 → 턴어라운드 초기 가능

3단계: 이익가속/감속으로 방향성 확인
  └── 가속 → 성장 가속 구간 (강력 매수 시그널)
  └── 감속 → 성장 피크 가능 (리스크 관리)

4단계: 연속 성장 카운터로 지속성 확인
  └── 3년+ 연속 성장 → 구조적 성장주
  └── 0~1년 → 변동성 높은 실적, 일회성 가능
```

### 7.2 성장주 선별 기준 (시스템 내 고성장 전략)

```
[하드 필터]
✓ 매출 CAGR ≥ 10%
✓ 영업이익 CAGR ≥ 10%
✓ 분기 영업이익 YoY > 0%
✓ RS등급 ≥ 50
✓ TTM 영업CF > 0
✓ 시가총액 ≥ 500억원

[소프트 스코어링]
분기 영업이익 YoY    20%
이익가속 신호        20%
영업이익 CAGR       15%
RS등급              25%
PEG Inverse        20%
```

### 7.3 성장의 질(Quality) 점검 체크리스트

| 점검 항목 | 양호 | 주의 |
|----------|------|------|
| 매출 동반 성장 | 매출↑ + 이익↑ | 이익↑ but 매출↓ |
| 현금흐름 뒷받침 | 영업CF > 순이익 | 영업CF < 순이익 |
| 마진 추세 | 영업이익률 개선 | 매출만 성장, 마진 하락 |
| CAPEX 부담 | FCF 양수 | FCF 음수 (투자 과다) |
| 성장 가속 | YoY 성장률 증가 | YoY 성장률 감소 |

### 7.4 성장률 왜곡 주의사항

| 왜곡 유형 | 원인 | 대응 |
|----------|------|------|
| 기저효과 | 전년 극단적 저실적 | 2~3년 CAGR로 추세 확인 |
| 인수합병 효과 | M&A로 인한 외형 성장 | 유기적 성장률 별도 확인 |
| 환율 효과 | 원화 약세 시 수출 매출 증가 | 달러 기준 매출 추이 확인 |
| 일회성 이익 | 자산매각, 보험금 등 | TTM vs 연간 비교, CF 검증 |
| 회계 변경 | 기준 변경에 의한 착시 | 동일 기준 재계산 |

### 7.5 종합점수에서의 성장 가중치

```
종합점수 = 성장(30%) + 안정(30%) + 밸류에이션(30%) + 타이밍(10%)

성장 축 (30%) 내부:
├── 영업이익 CAGR:    35%
├── 매출 CAGR:       30%
├── 분기 영업이익 YoY: 25%
└── 이익가속:         10%
```
