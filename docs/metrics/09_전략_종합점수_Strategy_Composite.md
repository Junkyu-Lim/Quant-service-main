## 1. 스코어링 방법론

### 1.1 퍼센타일 랭킹 (get_rank)

모든 점수는 **퍼센타일 랭킹**으로 변환됩니다. 절대값이 아닌 전체 종목 대비 상대적 위치를 0~100으로 나타냅니다.

```python
# 오름차순 (높을수록 좋은 지표): asc=True
# 예: ROE, FCF Yield, RS등급
rank = percentile_rank(값, method='min') * 100

# 내림차순 (낮을수록 좋은 지표): asc=False
# 예: PER, PBR, 부채비율
rank = (1 - percentile_rank(값)) * 100
```

### 1.2 NaN 처리 모드

| 모드 | 설정 | 동작 | 적용 |
|------|------|------|------|
| **엄격 (Strict)** | zero_if_nan=True | NaN → 0점 (최하위) | 필수 지표 (PER, ROE 등) |
| **관대 (Lenient)** | zero_if_nan=False | NaN → 중앙값으로 채움 후 랭킹 | 보조 지표 |

### 1.3 과열 감도 적용

모든 전략 점수에는 과열 감도가 적용됩니다:

```
최종 전략점수 = 원점수 × (1 - clip((과열도-40)/60, 0, 1) × 감도)
```

과열도 40 이하에서는 페널티 없음. 40 초과 시 선형 증가, 100 도달 시 최대 페널티.

## 2. 전략 1: 주도주 (Market Leaders)

### 투자 철학
윌리엄 오닐(William O'Neil)의 CAN SLIM 방법론을 기반으로, **시장을 선도하는 강한 모멘텀 종목**을 선별합니다. "강한 주식은 더 강해진다"는 원칙입니다.

### 하드 필터 (진입 조건)

```
✓ 시가총액 ≥ 1조원 (100,000,000,000)
✓ TTM 순이익 > 0
✓ RS등급 ≥ 80 (시장 상위 20%)
✓ 20일 평균 거래대금 > 5억원 (500,000,000)
✓ 수급강도 > 0
```

> webapp `_apply_screen_filter("leaders")` 와 동일한 조건 (`app.py` Lines 1403~1417)

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| RS등급 | 25% | ↑ | 시장 대비 상대강도 |
| 수급강도 | 20% | ↑ | 외국인+기관 자금 유입 |
| 20일 평균 거래대금 | 10% | ↑ | 유동성 |
| 영업이익 CAGR | 15% | ↑ | 이익 성장력 |
| 분기 영업이익 YoY | 15% | ↑ | 최근 이익 모멘텀 |
| 이익가속 | 10% | ↑ | 성장 가속 여부 |
| RSI_14 | 5% | ↑ | 기술적 모멘텀 |

### 과열 감도: **0.15** (가장 낮음)
→ 모멘텀 전략이므로 과열에 상대적으로 강함

### 적합한 투자자
- 추세 추종 투자자
- 단기~중기 수익 추구
- 적극적 리스크 관리 가능한 투자자

## 3. 전략 2: 우량가치 (Quality Value)

### 투자 철학
조엘 그린블라트(Joel Greenblatt)의 **마법 공식**과 피터 린치(Peter Lynch)의 **GARP** 전략을 결합하여, 높은 자본수익률 + 합리적 가격의 기업을 선별합니다.

### Track A: 일반 기업 (제조/서비스)

#### 하드 필터

```
✓ ROIC ≥ 10% (그린블라트 해자 기준)
✓ F-Score ≥ 5 (피오트로스키 품질)
✓ PEG < 1.2 (린치 성장-가치 균형)
✓ 부채비율 < 120%
✓ 유동비율 > 120%
✓ 금기 순이익 > 0
✓ 전기 순이익 > 0 (음수 불가)
✓ 시가총액 ≥ 1,000억원 (100,000,000,000)
✓ 가치함정 경고 = 0
```

### Track B: 금융/지주사

금융업은 자본구조가 달라 별도 기준을 적용합니다.

> 판별 기준: 종목명에 "지주|금융|은행|증권|생명|화재" 포함 OR 유동비율 데이터 없음

#### 하드 필터

```
✓ ROE ≥ 8% (ROIC 대신, 금융업 특성)
✓ PBR < 1.5 (한국 금융주 노멀)
✓ 배당수익률 ≥ 2% (주주환원)
✓ F-Score ≥ 4 (금융업 회계 특성 감안, 완화)
✓ 금기 순이익 > 0
✓ 전기 순이익 > 0 (음수 불가)
✓ 시가총액 ≥ 3,000억원 (300,000,000,000)
✓ 가치함정 경고 = 0
```

### 스코어링 (Track A & B 공통)

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| FCF 수익률 | 20% | ↑ | 현금흐름 수익률 |
| ROIC | 20% | ↑ | 투하자본 수익률 (해자) |
| 괴리율 | 20% | ↑ | S-RIM 대비 저평가도 |
| F-Score | 15% | ↑ | 재무건전성 |
| 지속가치_품질 | 15% | ↑ | 복합 품질 스코어 (0~6점, `03_수익성` 참고) |
| 매출이익_동행성 | 10% | ↑ | 매출·이익 함께 증가 여부 |

> ⚠️ **구버전 문서 대비 변경**: PEG역수(10%) 제거 → 지속가치_품질(15%), 매출이익_동행성(10%) 추가. FCF/ROIC 비중 25%→20%로 조정.

### 과열 감도: **0.35** (가장 높음)
→ 가치투자는 싸게 사야 하므로 과열에 가장 민감

### 적합한 투자자
- 장기 가치투자자
- 안정적 수익 추구
- 버핏/그린블라트 스타일

## 4. 전략 3: 고성장 (High Growth)

### 투자 철학
**높은 성장률**과 **가속하는 이익 모멘텀**을 가진 기업을 선별합니다. 성장의 질을 현금흐름으로 검증합니다.

### 하드 필터

```
✓ 매출 CAGR ≥ 10%
✓ 영업이익 CAGR ≥ 10%
✓ 분기 영업이익 YoY > 0%
✓ RS등급 ≥ 50 (시장 평균 이상)
✓ TTM 영업CF > 0 (현금흐름으로 뒷받침)
✓ 시가총액 ≥ 500억원 (50,000,000,000)
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| RS등급 | 25% | ↑ | 시장 대비 모멘텀 |
| 분기 영업이익 YoY | 20% | ↑ | 최근 분기 이익 성장 |
| 이익가속 | 20% | ↑ | 성장 가속 중인지 |
| PEG 역수 | 20% | ↑ | 성장 대비 밸류에이션 |
| 영업이익 CAGR | 15% | ↑ | 중장기 이익 성장률 |

### 과열 감도: **0.20**
→ 성장주는 펀더멘탈로 지지되지만 밸류에이션에 민감

### 적합한 투자자
- 성장주 투자자
- 중기 수익 추구 (6개월~2년)
- 실적 발표 기반 매매

## 5. 전략 4: 현금배당 (Cash Dividend)

### 투자 철학
**지속가능한 배당 성장**과 **높은 FCF 수익률**을 가진 기업을 선별합니다. 단순 고배당이 아닌, 배당의 질과 지속가능성을 중시합니다.

### 하드 필터

```
✓ FCF Yield ≥ 3%
✓ 배당수익률 ≥ 1%
✓ 부채비율 < 120%
✓ 시가총액 ≥ 500억원 (50,000,000,000)
✓ 배당성향 < 80% (지속가능성)
✓ 현금전환율 ≥ 70% (이익의 질)
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| FCF Yield | 25% | ↑ | 잉여현금흐름 수익률 |
| 배당수익률 | 20% | ↑ | 현재 배당 수익률 |
| DPS CAGR | 15% | ↑ | 배당금 성장률 |
| ROIC Moat | 15% | ↑ | 수익 창출 능력 (해자) |
| 배당성향 역수 | 10% | ↓ | 낮을수록 여유 있음 |
| F-Score | 10% | ↑ | 재무건전성 |
| 부채비율 역수 | 5% | ↓ | 낮을수록 안전 |

### 보너스 & 페널티

```
[보너스]
+ 배당 연속 성장 보너스: min(log₂(연수+1) × 3, 10)
+ 배당+이익 동반성장: +2점 (이익≥2년 + 배당≥1년 연속 성장)

[페널티]
× 밸류트랩 경고 시: ×0.7 (30% 감점)
  └── 조건: 배당성향>80% OR (배당수익률>10% AND RS<30) OR 현금전환율<70%
```

### 과열 감도: **0.30**
→ 금리 민감, 채권 대체재 성격

### 적합한 투자자
- 인컴 투자자
- 장기 배당 재투자 전략
- 은퇴 자금 운용

## 6. 전략 5: 턴어라운드 (Turnaround)

### 투자 철학
적자→흑자 전환 또는 수익성 급개선 기업을 선별합니다. **질적 전환이 확인된** 기업에 한해 투자하여, 무분별한 저가매수를 방지합니다.

### 하드 필터

```
✓ (흑자전환 플래그 = 1) OR (이익률_급개선 = 1)
  └── 흑자전환: 전기 순이익 < 0 AND 금기 순이익 > 0
  └── 이익률_급개선: 영업이익률 5%p 이상 개선
✓ TTM 순이익 > 0 (흑자 확인)
✓ TTM 영업CF > 0 (현금흐름 검증)
✓ 분기 매출 YoY > -15% (구조적 쇠퇴 방지)
✓ 시가총액 ≥ 300억원 (30,000,000,000)
✓ 이자보상배율 > 1.5 (파산 리스크 배제)
✓ 스마트머니 승률 ≥ 50% OR VCP 신호 = 1 (데이터 없을 시 통과)
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| 흑자전환 플래그 | 15% | ↑ | 적자→흑자 전환 |
| 스마트머니 승률 | 15% | ↑ | 기관/외국인 매수 일관성 |
| 분기 매출 YoY | 15% | ↑ | 매출 성장 (구조적 회복) |
| 퀄리티 턴어라운드 | 15% | ↑ | GPM↑ + CF↑ + ROIC↑ 복합 |
| 순이익률 변화 | 10% | ↑ | 마진 개선 폭 |
| GPM 변화 | 10% | ↑ | 매출총이익률 개선 |
| 이자보상배율 | 10% | ↑ | 재무 안전성 |
| 괴리율 | 10% | ↑ | S-RIM 대비 저평가도 |

### 과열 감도: **0.25**
→ 턴어라운드 초기 종목이 과열되면 위험

### 적합한 투자자
- 역발상 투자자
- 실적 변곡점 포착 능력
- 중기 수익 추구

## 7. 종합점수 (4축 모델)

### 정의
모든 지표를 4개 축으로 재구성하여 **전략에 무관한 종합적 매력도**를 산출합니다.

### 산출 공식

```
종합점수 = (성장 × 0.25) + (안정 × 0.35) + (가격 × 0.30) + (타이밍 × 0.15)
```

> ⚠️ **구버전 대비 변경**: 성장 30%→25%, 안정 30%→35%, 타이밍 10%→15%

---

### 축 1: 성장 (25%)

```python
성장축 = (영업이익_CAGR_rank × 0.30)
       + (매출_CAGR_rank    × 0.25)
       + (Q_영업이익_YoY_rank × 0.25)
       + (이익가속_rank       × 0.10)
       + (영업이익_연속성장_rank × 0.10)
```

| 구성요소 | 비중 |
|---------|------|
| 영업이익 CAGR | 30% |
| 매출 CAGR | 25% |
| 분기 영업이익 YoY | 25% |
| 이익가속 연속 | 10% |
| 영업이익 연속성장 연수 | 10% |

**Base Effect Penalty**: CAGR이 상한(150%)에 도달 AND YoY도 상한(300%)에 도달 AND 전기 영업이익률 < 3% 를 모두 충족하면 성장축 점수 × 0.75 적용 (베이스효과 왜곡 보정)

---

### 축 2: 안정 (35%)

```python
안정축 = (F스코어_rank     × 0.30)
       + (지속가치_품질_rank × 0.25)
       + (FCF수익률_rank    × 0.20)
       + (ROE_rank          × 0.15)
       + (이자보상배율_rank   × 0.10)
```

| 구성요소 | 비중 |
|---------|------|
| F-Score | 30% |
| 지속가치_품질 (0~6점) | 25% |
| FCF Yield | 20% |
| ROE | 15% |
| 이자보상배율 | 10% |

**Debt Penalty**: 부채비율 > 200% 시 안정축 점수 × 0.80

> ⚠️ **구버전 대비 변경**: ROE 40%→15%, F-Score 35%→30%, FCF 25%→20%. 지속가치_품질(25%), 이자보상배율(10%) 신규 추가.

---

### 축 3: 가격 (30%)

```python
가격축 = (PER역수_rank    × 0.35)
       + (괴리율_rank      × 0.25)
       + (PBR역수_rank    × 0.20)
       + (FCF_rank         × 0.10)
       + (밸류_모델수_rank  × 0.10)
```

| 구성요소 | 비중 |
|---------|------|
| PER 역수 | 35% |
| 괴리율 (S-RIM 대비) | 25% |
| PBR 역수 | 20% |
| TTM FCF | 10% |
| 밸류 모델수 | 10% |

**Value Trap Penalty**: 가치함정_경고 = 1 시 가격축 점수 × 0.60 (40% 감점)

**Base Effect 보정**: 베이스효과 조건 충족 시 괴리율 기여 × 0.5 (S-RIM 고평가 왜곡 보정)

> 밸류_모델수: 적정주가 산출이 가능한 모델(S-RIM, DDM, PER, PBR 등) 수. 많을수록 신뢰도 높음.

---

### 축 4: 타이밍 (15%)

```python
_base = (비과열_rank  × 0.30)   # 100 - 과열도
      + (돌파신호_rank × 0.35)
      + (RS등급_rank   × 0.20)
      + (수급강도_rank  × 0.15)

_decel = -15 if 실적감속_경고 == 1 else 0
타이밍축 = clip(_base + _decel, 0, 100)
```

| 구성요소 | 비중 |
|---------|------|
| 비과열 (100 - 과열도) | 30% |
| 돌파신호 | 35% |
| RS등급 | 20% |
| 수급강도 | 15% |
| 실적감속 페널티 | -15점 (감속 경고 시) |

> ⚠️ **구버전 대비 변경**: RS등급(20%), 수급강도(15%) 신규 추가. 비과열 40%→30%, 돌파신호 45%→35%.

---

### 종합점수 해석

| 종합점수 | 등급 | 해석 |
|---------|------|------|
| 80 ~ 100 | S | 전 영역에서 우수, 최우선 관심 |
| 65 ~ 80 | A | 대부분 영역 양호 |
| 50 ~ 65 | B | 평균 이상 |
| 35 ~ 50 | C | 평균 이하, 선별적 접근 |
| 20 ~ 35 | D | 부진 |
| 0 ~ 20 | F | 대부분 지표 부정적 |

### 8.1 전략 선택 가이드

### 8.3 종합점수 vs 전략점수 활용법

| 상황 | 활용 지표 | 이유 |
|------|----------|------|
| 전체 유니버스에서 상위 종목 선별 | 종합점수 | 편향 없는 전체 평가 |
| 특정 스타일 종목 선별 | 전략점수 | 스타일 특화 평가 |
| 보유 종목 리밸런싱 | 전략점수 + 과열점수 | 전략 적합성 + 리스크 |
| 섹터 내 비교 | 종합점수 | 동일 조건 비교 가능 |
| 매매 타이밍 | 과열/돌파 + 타이밍 축 | 시점 판단 특화 |

### 8.4 전략별 리밸런싱 주기 가이드

| 전략 | 권장 리밸런싱 | 이유 |
|------|------------|------|
| 주도주 | 2~4주 | 모멘텀 변화 빠름 |
| 고성장 | 분기 (실적 발표 후) | 분기 실적이 핵심 트리거 |
| 우량가치 | 반기~연간 | 가치 실현에 시간 필요 |
| 현금배당 | 반기~연간 | 배당 정책 변경 드묾 |
| 턴어라운드 | 분기 (실적 확인) | 전환 지속 여부 모니터링 |

### 8.5 전략 점수 해석 시 주의사항

| 주의사항 | 설명 |
|---------|------|
| 절대 점수 비교 금지 | 전략 간 점수는 비교 불가 (주도주 70점 ≠ 우량가치 70점) |
| 과열 감도 인지 | 같은 과열이라도 전략별 영향 다름 (우량가치 최대 -35%) |
| 하드 필터 통과 전제 | 스코어는 필터 통과 종목에만 의미 |
| 시장 국면 고려 | 약세장에서 주도주 점수는 전반적으로 낮아짐 |
| 섹터 편향 | 특정 전략이 특정 섹터에 편중될 수 있음 (배당→금융 등) |

### 8.6 5대 전략 한눈에 비교

| | 주도주 | 우량가치 | 고성장 | 현금배당 | 턴어라운드 |
|---|---|---|---|---|---|
| **핵심 철학** | 모멘텀 | 가치+품질 | 성장 | 배당+안전 | 전환 |
| **시총 기준** | ≥1조 | ≥1000억/3000억 | ≥500억 | ≥500억 | ≥300억 |
| **과열 감도** | 0.15 | 0.35 | 0.20 | 0.30 | 0.25 |
| **핵심 지표** | RS등급 | ROIC, FCF | CAGR, YoY | FCF Yield | 턴어라운드 |
| **투자 기간** | 단기~중기 | 장기 | 중기 | 장기 | 중기 |
| **리밸런싱** | 2~4주 | 반기 | 분기 | 반기 | 분기 |
| **참고 인물** | 오닐 | 그린블라트, 린치 | - | - | - |
