## 1. 스코어링 방법론

### 1.1 퍼센타일 랭킹 (get_rank)

모든 점수는 **퍼센타일 랭킹**으로 변환됩니다. 절대값이 아닌 전체 종목 대비 상대적 위치를 0~100으로 나타냅니다.

```python
# 오름차순 (높을수록 좋은 지표): asc=True
# 예: ROE, FCF Yield, RS등급
rank = percentile_rank(값, method='min') * 100

# 내림차순 (낮을수록 좋은 지표): asc=False
# 예: PER, PBR, 부채비율
rank = (1 - percentile_rank(값)) * 100
```

### 1.2 NaN 처리 모드

| 모드 | 설정 | 동작 | 적용 |
|------|------|------|------|
| **엄격 (Strict)** | zero_if_nan=True | NaN → 0점 (최하위) | 필수 지표 (PER, ROE 등) |
| **관대 (Lenient)** | zero_if_nan=False | NaN → 중앙값으로 채움 후 랭킹 | 보조 지표 |

### 1.3 과열 감도 적용

모든 전략 점수에는 과열 감도가 적용됩니다:

```
최종 전략점수 = 원점수 × (1 - clip((과열-40)/60, 0, 1) × 감도)
```

## 2. 전략 1: 주도주 (Market Leaders)

### 투자 철학
윌리엄 오닐(William O'Neil)의 CAN SLIM 방법론을 기반으로, **시장을 선도하는 강한 모멘텀 종목**을 선별합니다. "강한 주식은 더 강해진다"는 원칙입니다.

### 하드 필터 (진입 조건)

```
✓ 시가총액 ≥ 1조원
✓ TTM 영업이익 > 0
✓ RS등급 ≥ 80 (시장 상위 20%)
✓ 20일 평균 거래대금 > 5억원
✓ 수급강도 > 0
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| RS등급 | 25% | ↑ | 시장 대비 상대강도 |
| 수급강도 | 20% | ↑ | 외국인+기관 자금 유입 |
| 20일 평균 거래대금 | 10% | ↑ | 유동성 |
| 영업이익 CAGR | 15% | ↑ | 이익 성장력 |
| 분기 영업이익 YoY | 15% | ↑ | 최근 이익 모멘텀 |
| 이익가속 | 10% | ↑ | 성장 가속 여부 |
| RSI_14 | 5% | ↑ | 기술적 모멘텀 |

### 정렬: 주도주 점수 (내림차순)

### 과열 감도: **0.15** (가장 낮음)
→ 모멘텀 전략이므로 과열에 상대적으로 강함

### 적합한 투자자
- 추세 추종 투자자
- 단기~중기 수익 추구
- 적극적 리스크 관리 가능한 투자자

## 3. 전략 2: 우량가치 (Quality Value)

### 투자 철학
조엘 그린블라트(Joel Greenblatt)의 **마법 공식**과 피터 린치(Peter Lynch)의 **GARP** 전략을 결합하여, 높은 자본수익률 + 합리적 가격의 기업을 선별합니다.

### Track A: 일반 기업 (제조/서비스)

#### 하드 필터

```
✓ ROIC ≥ 10% (그린블라트 해자 기준)
✓ F-Score ≥ 5 (피오트로스키 품질)
✓ PEG < 1.2 (린치 성장-가치 균형)
✓ 부채비율 < 120%
✓ 유동비율 > 120%
✓ 금기 순이익 > 0
✓ 전기 순이익 > 0 (음수 불가)
✓ 시가총액 ≥ 1,000억원
```

### Track B: 금융/지주사

금융업은 자본구조가 달라 별도 기준을 적용합니다.

#### 하드 필터

```
✓ ROE ≥ 8% (ROIC 대신, 금융업 특성)
✓ PBR < 1.5 (한국 금융주 노멀)
✓ 배당수익률 ≥ 2% (주주환원)
✓ F-Score ≥ 4 (금융업 회계 특성 감안, 완화)
✓ 금기 순이익 > 0
✓ 전기 순이익 > 0 (음수 불가)
✓ 시가총액 ≥ 3,000억원
```

### 스코어링 (Track A & B 공통)

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| FCF Yield | 25% | ↑ | 현금흐름 수익률 |
| ROIC | 25% | ↑ | 투하자본 수익률 (해자) |
| F-Score | 20% | ↑ | 재무건전성 |
| 괴리율 | 20% | ↑ | S-RIM 대비 저평가도 |
| PEG 역수 | 10% | ↑ | 성장 대비 가치 |

### 정렬: 우량가치 점수 (내림차순)

### 과열 감도: **0.35** (가장 높음)
→ 가치투자는 싸게 사야 하므로 과열에 가장 민감

### 적합한 투자자
- 장기 가치투자자
- 안정적 수익 추구
- 버핏/그린블라트 스타일

## 4. 전략 3: 고성장 (High Growth)

### 투자 철학
**높은 성장률**과 **가속하는 이익 모멘텀**을 가진 기업을 선별합니다. 성장의 질을 현금흐름으로 검증합니다.

### 하드 필터

```
✓ 매출 CAGR ≥ 10%
✓ 영업이익 CAGR ≥ 10%
✓ 분기 영업이익 YoY > 0%
✓ RS등급 ≥ 50 (시장 평균 이상)
✓ TTM 영업CF > 0 (현금흐름으로 뒷받침)
✓ 시가총액 ≥ 500억원
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| 분기 영업이익 YoY | 20% | ↑ | 최근 분기 이익 성장 |
| 이익가속 | 20% | ↑ | 성장 가속 중인지 |
| 영업이익 CAGR | 15% | ↑ | 중장기 이익 성장률 |
| RS등급 | 25% | ↑ | 시장 대비 모멘텀 |
| PEG 역수 | 20% | ↑ | 성장 대비 밸류에이션 |

### 정렬: 고성장 점수 (내림차순)

### 과열 감도: **0.20**
→ 성장주는 펀더멘탈로 지지되지만 밸류에이션에 민감

### 적합한 투자자
- 성장주 투자자
- 중기 수익 추구 (6개월~2년)
- 실적 발표 기반 매매

## 5. 전략 4: 현금배당 (Cash Dividend)

### 투자 철학
**지속가능한 배당 성장**과 **높은 FCF 수익률**을 가진 기업을 선별합니다. 단순 고배당이 아닌, 배당의 질과 지속가능성을 중시합니다.

### 하드 필터

```
✓ FCF Yield ≥ 3%
✓ 배당수익률 ≥ 1%
✓ 부채비율 < 120%
✓ 시가총액 ≥ 500억원
✓ 배당성향 < 80% (지속가능성)
✓ 현금전환율 ≥ 70% (이익의 질)
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| FCF Yield | 25% | ↑ | 잉여현금흐름 수익률 |
| 배당수익률 | 20% | ↑ | 현재 배당 수익률 |
| DPS CAGR | 15% | ↑ | 배당금 성장률 |
| ROIC Moat | 15% | ↑ | 수익 창출 능력 (해자) |
| 배당성향 역수 | 10% | ↓ | 낮을수록 여유 있음 |
| F-Score | 10% | ↑ | 재무건전성 |
| 부채비율 역수 | 5% | ↓ | 낮을수록 안전 |

### 보너스 & 페널티

```
[보너스]
+ 배당 연속 성장 보너스: min(log₂(연수+1) × 3, 10)
+ 배당+이익 동반성장: +2점 (이익≥2년 + 배당≥1년 연속 성장)

[페널티]
× 밸류트랩 경고 시: ×0.7 (30% 감점)
```

### 정렬: 현금배당 점수 (내림차순)

### 과열 감도: **0.30**
→ 금리 민감, 채권 대체재 성격

### 적합한 투자자
- 인컴 투자자
- 장기 배당 재투자 전략
- 은퇴 자금 운용

## 6. 전략 5: 턴어라운드 (Turnaround)

### 투자 철학
적자→흑자 전환 또는 수익성 급개선 기업을 선별합니다. **질적 전환이 확인된** 기업에 한해 투자하여, 무분별한 저가매수를 방지합니다.

### 하드 필터

```
✓ (턴어라운드 플래그 = 1) OR (마진 급개선 = 1)
  └── 흑자전환 또는 영업이익률 5%p 이상 개선
✓ TTM 순이익 > 0 (흑자 확인)
✓ TTM 영업CF > 0 (현금흐름 검증)
✓ 분기 매출 YoY > -15% (구조적 쇠퇴 방지)
✓ 시가총액 ≥ 300억원
✓ 이자보상배율 > 1.5 (파산 리스크 배제)
```

### 스코어링

| 지표 | 비중 | 방향 | 설명 |
|------|------|------|------|
| 순이익률 변화 | 10% | ↑ | 마진 개선 폭 |
| 턴어라운드 플래그 | 15% | ↑ | 적자→흑자 전환 |
| 스마트머니 승률 | 15% | ↑ | 기관/외국인 매수 일관성 |
| GPM 변화 | 10% | ↑ | 매출총이익률 개선 |
| 분기 매출 YoY | 15% | ↑ | 매출 성장 (구조적 회복) |
| 이자보상배율 | 10% | ↑ | 재무 안전성 |
| 퀄리티 턴어라운드 | 15% | ↑ | GPM↑ + CF↑ + ROIC↑ 복합 |
| 괴리율 | 10% | ↑ | S-RIM 대비 저평가도 |

### 정렬: 턴어라운드 점수 (내림차순)

### 과열 감도: **0.25**
→ 턴어라운드 초기 종목이 과열되면 위험

### 적합한 투자자
- 역발상 투자자
- 실적 변곡점 포착 능력
- 중기 수익 추구

## 7. 종합점수 (4축 모델)

### 정의
모든 지표를 4개 축으로 재구성하여 **전략에 무관한 종합적 매력도**를 산출합니다.

### 산출 공식

```
종합점수 = (성장 × 0.30) + (안정 × 0.30) + (밸류에이션 × 0.30) + (타이밍 × 0.10)
```

### 축 1: 성장 (30%)

```
성장 = (영업이익CAGR_rank × 0.35)
     + (매출CAGR_rank × 0.30)
     + (분기영업이익YoY_rank × 0.25)
     + (이익가속_rank × 0.10)
```

| 구성요소 | 비중 | 방향 |
|---------|------|------|
| 영업이익 CAGR | 35% | ↑ |
| 매출 CAGR | 30% | ↑ |
| 분기 영업이익 YoY | 25% | ↑ |
| 이익가속 | 10% | ↑ |

### 축 2: 안정 (30%)

```
안정 = (ROE_rank × 0.40)
     + (F_Score_rank × 0.35)
     + (FCF_Yield_rank × 0.25)
```

| 구성요소 | 비중 | 방향 |
|---------|------|------|
| ROE | 40% | ↑ |
| F-Score | 35% | ↑ |
| FCF Yield | 25% | ↑ |

### 축 3: 밸류에이션 (30%)

```
밸류에이션 = (PER역수_rank × 0.40)
           + (괴리율_rank × 0.35)
           + (PBR역수_rank × 0.25)
```

| 구성요소 | 비중 | 방향 |
|---------|------|------|
| PER 역수 | 40% | ↑ (PER 낮을수록 좋음) |
| 괴리율 | 35% | ↑ (양수 = 저평가) |
| PBR 역수 | 25% | ↑ (PBR 낮을수록 좋음) |

### 축 4: 타이밍 (10%)

```
_base = (비과열_rank × 0.40) + (돌파신호_rank × 0.45)
_decel = -15 if 이익감속경고 else 0
타이밍 = clip((_base + _decel) / 0.85, 0, 100)
```

| 구성요소 | 비중 | 방향 |
|---------|------|------|
| 비과열 (100-과열점수) | 40% | ↑ (과열 안 될수록 좋음) |
| 돌파신호 | 45% | ↑ |
| 이익감속 페널티 | -15점 | ↓ (감속 시 감점) |
| 정규화 | ÷0.85 | 범위 조정 |

### 종합점수 해석

| 종합점수 | 등급 | 해석 |
|---------|------|------|
| 80 ~ 100 | S | 전 영역에서 우수, 최우선 관심 |
| 65 ~ 80 | A | 대부분 영역 양호 |
| 50 ~ 65 | B | 평균 이상 |
| 35 ~ 50 | C | 평균 이하, 선별적 접근 |
| 20 ~ 35 | D | 부진 |
| 0 ~ 20 | F | 대부분 지표 부정적 |

## 8. 실전 활용 가이드

### 8.1 전략 선택 가이드

```
[시장 국면별 전략 선택]

강세장 (KOSPI 상승 추세):
├── 1순위: 주도주 — 모멘텀 극대화
├── 2순위: 고성장 — 성장주 프리미엄 확대
└── 3순위: 턴어라운드 — 시장 상승에 편승

약세장 (KOSPI 하락 추세):
├── 1순위: 우량가치 — 안전마진 확보
├── 2순위: 현금배당 — 배당으로 방어
└── 3순위: (현금 비중 확대)

박스권 (횡보):
├── 1순위: 현금배당 — 배당수익으로 수익 확보
├── 2순위: 우량가치 — 저평가 종목 발굴
└── 3순위: 턴어라운드 — 개별 종목 이벤트
```

### 8.2 전략 조합 포트폴리오

```
[균형 포트폴리오 예시]

보수적 포트폴리오:
├── 우량가치:  40%
├── 현금배당:  30%
├── 고성장:   20%
└── 주도주:   10%

공격적 포트폴리오:
├── 주도주:   30%
├── 고성장:   30%
├── 턴어라운드: 20%
└── 우량가치:  20%

인컴 포트폴리오:
├── 현금배당:  50%
├── 우량가치:  30%
└── 고성장:   20%
```

### 8.3 종합점수 vs 전략점수 활용법

| 상황 | 활용 지표 | 이유 |
|------|----------|------|
| 전체 유니버스에서 상위 종목 선별 | 종합점수 | 편향 없는 전체 평가 |
| 특정 스타일 종목 선별 | 전략점수 | 스타일 특화 평가 |
| 보유 종목 리밸런싱 | 전략점수 + 과열점수 | 전략 적합성 + 리스크 |
| 섹터 내 비교 | 종합점수 | 동일 조건 비교 가능 |
| 매매 타이밍 | 과열/돌파 + 타이밍 축 | 시점 판단 특화 |

### 8.4 전략별 리밸런싱 주기 가이드

| 전략 | 권장 리밸런싱 | 이유 |
|------|------------|------|
| 주도주 | 2~4주 | 모멘텀 변화 빠름 |
| 고성장 | 분기 (실적 발표 후) | 분기 실적이 핵심 트리거 |
| 우량가치 | 반기~연간 | 가치 실현에 시간 필요 |
| 현금배당 | 반기~연간 | 배당 정책 변경 드묾 |
| 턴어라운드 | 분기 (실적 확인) | 전환 지속 여부 모니터링 |

### 8.5 전략 점수 해석 시 주의사항

| 주의사항 | 설명 |
|---------|------|
| 절대 점수 비교 금지 | 전략 간 점수는 비교 불가 (주도주 70점 ≠ 우량가치 70점) |
| 과열 감도 인지 | 같은 과열이라도 전략별 영향 다름 (우량가치 최대 -35%) |
| 하드 필터 통과 전제 | 스코어는 필터 통과 종목에만 의미 |
| 시장 국면 고려 | 약세장에서 주도주 점수는 전반적으로 낮아짐 |
| 섹터 편향 | 특정 전략이 특정 섹터에 편중될 수 있음 (배당→금융 등) |
