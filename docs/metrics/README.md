## 각 문서 구성

모든 문서는 다음 구조를 따릅니다:

1. **정의**: 지표의 개념과 목적
2. **산출 공식**: 정확한 계산 로직과 파라미터
3. **해석 기준**: 범위별 의미와 등급
4. **시스템 내 활용**: 전략/종합점수에서의 역할과 비중
5. **실전 활용 가이드**: 분석 프레임워크, 조합 전략, 주의사항
