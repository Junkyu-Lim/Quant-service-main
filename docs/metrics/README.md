## 종합점수 4축 구조 (한눈에 보기)

```
종합점수 = 성장(25%) + 안정(35%) + 가격(30%) + 타이밍(15%)

성장 (25%)            안정 (35%)            가격 (30%)             타이밍 (15%)
├── 영업이익CAGR 30%  ├── F-Score      30%  ├── PER역수     35%   ├── 돌파신호   35%
├── 매출CAGR    25%  ├── 지속가치_품질 25%  ├── 괴리율      25%   ├── 비과열     30%
├── Q영업YoY   25%  ├── FCF Yield    20%  ├── PBR역수     20%   ├── RS등급     20%
├── 이익가속    10%  ├── ROE          15%  ├── TTM FCF     10%   └── 수급강도   15%
└── 연속성장    10%  └── 이자보상배율  10%  └── 밸류모델수   10%
                                                                   (-15점: 이익감속 시)
```

**패널티**:
- 성장축: Base Effect 조건 충족 시 × 0.75
- 안정축: 부채비율 > 200% 시 × 0.80
- 가격축: 가치함정_경고 = 1 시 × 0.60

## 5대 전략 필터 요약

| 전략 | 최소 시총 | 핵심 진입 조건 | 과열 감도 |
|-----|---------|------------|---------|
| **주도주** | 1조 | RS≥80, 거래대금>5억, 수급강도>0 | 0.15 |
| **우량가치 (일반)** | 1,000억 | ROIC≥10%, F≥5, PEG<1.2, 부채<120% | 0.35 |
| **우량가치 (금융)** | 3,000억 | ROE≥8%, PBR<1.5, 배당≥2%, F≥4 | 0.35 |
| **고성장** | 500억 | 매출·영업이익 CAGR≥10%, RS≥50 | 0.20 |
| **현금배당** | 500억 | FCF Yield≥3%, 배당수익률≥1%, 현금전환율≥70% | 0.30 |
| **턴어라운드** | 300억 | 흑자전환 OR 마진급개선, 이자보상>1.5 | 0.25 |

## 각 문서 구성

모든 문서는 다음 구조를 따릅니다:

1. **정의**: 지표의 개념과 목적
2. **산출 공식**: 정확한 계산 로직과 파라미터 (코드 기준)
3. **해석 기준**: 범위별 의미와 등급
4. **시스템 내 활용**: 전략/종합점수에서의 역할과 비중
5. **실전 활용 가이드**: 분석 프레임워크, 조합 전략, 주의사항
