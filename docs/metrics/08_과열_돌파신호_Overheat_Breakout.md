## 1. 과열점수 (Overheat Score)

### 정의
종목의 단기 과열 정도를 0~100 사이로 수치화한 복합 지표입니다. 높을수록 단기 조정 위험이 큽니다.

### 산출 공식 (4개 요소 가중합)

```
과열점수 = (52주근접 × 0.30) + (MA20과열 × 0.25) + (RSI과열 × 0.25) + (VPD × 0.20)
         + 이익감속 보너스

범위: 0 ~ 100+ (보너스 포함 시 100 초과 가능)
```

### 요소 1: 52주 고가 근접도 (30%)

```
기본값 = ((52주 고가 근접도(%) + 20) / 20) × 100
최종값 = clip(기본값, 0, 100)
```

| 52주 고가 대비 | 과열 점수 기여 |
|--------------|-------------|
| 0% (= 신고가) | 100점 |
| -5% | 75점 |
| -10% | 50점 |
| -20% | 0점 |
| -20% 이하 | 0점 |

### 요소 2: MA20 괴리 과열 (25%)

```
기본값 = (MA20 괴리율(%) / 20) × 100
최종값 = clip(기본값, 0, 100)
```

| MA20 괴리율 | 과열 점수 기여 |
|-----------|-------------|
| +20% 이상 | 100점 |
| +10% | 50점 |
| +5% | 25점 |
| 0% | 0점 |
| 음수 | 0점 |

### 요소 3: RSI 과매수 (25%)

```
기본값 = ((RSI - 50) / 30) × 100
최종값 = clip(기본값, 0, 100)
```

| RSI | 과열 점수 기여 |
|-----|-------------|
| ≥ 80 | 100점 |
| 70 | 67점 |
| 60 | 33점 |
| ≤ 50 | 0점 |

### 요소 4: 가격-거래량 다이버전스 (20%)

```
VPD = min(|주가 20일 수익률| + |거래대금 변화율|, 100)
조건: 주가 상승 + 거래량 감소인 경우에만 계산
```

주가는 오르는데 거래량이 빠지는 **상승 에너지 약화** 신호입니다.

### 이익감속 보너스

```
if 이익감속경고 = 1:
    과열점수 = clip(과열점수 + 10, 0, 100)
```

기술적 과열 + 펀더멘탈 악화가 겹치면 추가 위험 반영합니다.

**이익감속 경고 3가지 유형** (자세한 내용은 `02_성장성` 참고):

| 유형 | 조건 |
|------|------|
| 피크 이익형 | 최근 3분기 YoY 모두 양수 AND 단조 감소 |
| 손실 악화형 | 최근 2분기 연속 음수 AND 더 악화 |
| 흑자→적자 전환형 | 직전 분기 양수 → 최근 분기 음수 |

### 과열점수 종합 해석

| 과열점수 | 등급 | 해석 | 대응 |
|---------|------|------|------|
| 0 ~ 20 | 냉각 | 과열 없음, 안전 영역 | 매수 적합 |
| 20 ~ 40 | 정상 | 적정 수준 | 정상 운용 |
| 40 ~ 60 | 주의 | 일부 과열 징후 | 신규 매수 신중 |
| 60 ~ 80 | 경고 | 상당한 과열 | 비중 축소 고려 |
| 80 ~ 100+ | 위험 | 극심한 과열 | 차익 실현 또는 회피 |

## 2. 돌파신호 (Breakout Signal)

### 정의
종목이 새로운 상승 국면으로 돌파할 가능성을 0~100 사이로 수치화한 복합 지표입니다. 펀더멘탈 품질, 비과열 상태, 수급 유입, 가격 패턴의 4가지 축을 종합합니다.

### 산출 공식 (4개 축 가중합)

```
돌파신호 = (펀더멘탈 × 0.30) + (비과열 × 0.20) + (수급유입 × 0.30) + (가격패턴 × 0.20)
범위: 0 ~ 100 (소수 첫째 자리 반올림)
```

### 축 1: 펀더멘탈 품질 (30%)

```
펀더멘탈 = 이익가속(30) + ROIC개선(20) + GPM변화(15) + 턴어라운드(15) + F점수(20)
```

| 구성요소 | 점수 산출 | 최대 |
|---------|---------|------|
| 이익가속 | 이익가속 × 30 (0 or 1) | 30점 |
| ROIC 개선 | ROIC_개선 × 20 (0 or 1) | 20점 |
| GPM 변화 | clip(GPM_변화, 0, 15) | 15점 |
| 턴어라운드 | 턴어라운드 × 15 (0 or 1) | 15점 |
| F-Score | ≥7 → 20점, ≥5 → 10점, else 0 | 20점 |
| **합계** | | **100점** |

### 축 2: 비과열 (20%)

```
비과열 = ((70 - 과열점수) / 70) × 100
비과열 = clip(비과열, 0, 100)
```

| 과열점수 | 비과열 점수 |
|---------|----------|
| 0 | 100점 |
| 20 | 71점 |
| 40 | 43점 |
| 70 | 0점 |
| 100 | 0점 |

과열된 상태에서는 돌파 신뢰도가 떨어지므로 감점합니다.

### 축 3: 수급 유입 (30%)

```
수급유입 = 승률점수 + 쌍매수점수 + 수급강도점수 + 거래대금점수
```

| 구성요소 | 점수 산출 | 최대 |
|---------|---------|------|
| 스마트머니 승률 | clip((승률-0.5)/0.4×35, 0, 35) | 35점 |
| 쌍매수 비율 | clip((비율-0.2)/0.3×25, 0, 25) | 25점 |
| 수급강도 | 양수 시 clip(강도/2×20, 0, 20) | 20점 |
| 거래대금 변화 | 양수 시 clip(변화/50×20, 0, 20) | 20점 |
| **합계** | clip(합계, 0, 100) | **100점** |

### 축 4: 가격 패턴 (20%)

```
가격패턴 = VCP점수 + MA20근접점수 + RSI존점수
```

| 구성요소 | 점수 산출 | 최대 |
|---------|---------|------|
| VCP 신호 | VCP × 40 (0 or 1) | 40점 |
| MA20 근접 | 30 - \|괴리율-3\| × 3, clip 0~30 | 30점 |
| RSI 존 | 50~60→30, 40~50→15, 60~70→15, else 0 | 30점 |
| **합계** | | **100점** |

#### MA20 근접 점수 상세

```
이상적 위치: MA20 대비 +3% (살짝 위)
점수 = 30 - |MA20괴리율 - 3| × 3
```

| MA20 괴리율 | 점수 | 해석 |
|-----------|------|------|
| +3% | 30점 (최대) | 이상적 위치 |
| 0% | 21점 | MA에 걸쳐 있음 |
| +6% | 21점 | 약간 이격 |
| +10% | 9점 | 과열 방향 |
| +13% 이상 | 0점 | 너무 이격 |

#### RSI 존 점수 상세

| RSI 범위 | 점수 | 해석 |
|----------|------|------|
| 50 ~ 60 | 30점 (최대) | 가장 이상적 — 상승 초기, 아직 과매수 아님 |
| 40 ~ 50 | 15점 | 약세에서 회복 전환 가능 |
| 60 ~ 70 | 15점 | 이미 상승 중이지만 아직 여유 |
| 기타 | 0점 | 과매수/과매도 극단 |

### 돌파신호 종합 해석

| 돌파신호 | 등급 | 해석 |
|---------|------|------|
| 80 ~ 100 | **최강** | 모든 조건 충족, 돌파 확률 매우 높음 |
| 60 ~ 80 | **강** | 대부분 조건 양호, 적극 모니터링 |
| 40 ~ 60 | **보통** | 일부 조건 충족, 추가 확인 필요 |
| 20 ~ 40 | **약** | 조건 미흡, 관망 |
| 0 ~ 20 | **없음** | 돌파 가능성 낮음 |

## 3. 과열 감도 (전략별 적용)

### 정의
각 투자 전략이 과열 상태에 얼마나 민감하게 반응하는지를 결정하는 계수입니다.

### 과열 감도 매트릭스

| 전략 | 감도 | 과열 시 최대 감점 | 이유 |
|------|------|---------------|------|
| 주도주 | 0.15 | -15% | 모멘텀 전략이므로 과열에 상대적으로 강함 |
| 고성장 | 0.20 | -20% | 펀더멘탈 기반이지만 성장주는 밸류에이션 민감 |
| 턴어라운드 | 0.25 | -25% | 턴어라운드 초기 종목이 과열되면 위험 |
| 현금배당 | 0.30 | -30% | 금리 민감, 채권 대체재 성격 |
| 우량가치 | 0.35 | -35% | 가치투자는 싸게 사야 하므로 과열에 가장 민감 |

### 과열 감도 적용 공식

```
과열_유효값 = clip((과열점수 - 40) / 60, 0, 1)
전략점수_최종 = 전략점수_원점 × (1 - 과열_유효값 × 감도)
```

- **과열점수 ≤ 40**: 감점 없음 (정상 범위)
- **과열점수 = 70**: 과열_유효값 = 0.5
- **과열점수 = 100**: 과열_유효값 = 1.0 (최대 감점)

### 예시

```
우량가치 전략 원점수: 80점
과열점수: 85

과열_유효값 = (85 - 40) / 60 = 0.75
최종점수 = 80 × (1 - 0.75 × 0.35) = 80 × 0.7375 = 59.0점

→ 과열로 인해 80점 → 59점으로 하락
```

## 4. 실전 활용 가이드

### 4.1 과열점수 + 돌파신호 매트릭스

| | 돌파신호 높음 (≥60) | 돌파신호 보통 (30~60) | 돌파신호 낮음 (<30) |
|---|---|---|---|
| **과열 낮음 (<30)** | **최적 매수** | 매수 고려 | 관망 (모멘텀 부족) |
| **과열 보통 (30~60)** | 매수 가능 (비중 조절) | 관망 | 회피 |
| **과열 높음 (>60)** | 주의 (추격매수 위험) | **회피** | **절대 회피** |

### 4.2 돌파신호 4축 진단 활용법

```
[돌파 조건 점검 체크리스트]

✓ 펀더멘탈 (30%):
  ├── 이익가속 신호 활성화?
  ├── ROIC 전년 대비 개선?
  ├── 매출총이익률 개선 중?
  └── F-Score ≥ 5 이상?

✓ 비과열 (20%):
  ├── RSI < 70?
  ├── MA20 괴리율 < 10%?
  └── 52주 고가 대비 -10% 이상 이격?

✓ 수급 (30%):
  ├── 스마트머니 승률 ≥ 60%?
  ├── 쌍매수 비율 ≥ 20%?
  ├── 수급강도 양수?
  └── 거래대금 증가 추세?

✓ 가격패턴 (20%):
  ├── VCP 패턴 형성?
  ├── MA20 부근에서 거래?
  └── RSI 50~60 영역?
```

### 4.3 과열점수 활용 시나리오

#### 시나리오 A: 기존 보유 종목 리밸런싱

```
과열점수 > 70 + 이익감속 경고:
→ 비중 50% 이상 축소 권장
→ 특히 우량가치/배당 전략은 감도가 높아 점수 급락

과열점수 > 80:
→ 전략과 무관하게 차익 실현 검토
→ 추세 추종(주도주)도 리스크 관리 필요
```

#### 시나리오 B: 신규 매수 판단

```
돌파신호 ≥ 70 + 과열점수 < 40:
→ 적극 매수 영역
→ 펀더멘탈 + 수급 + 가격패턴 모두 정렬

돌파신호 ≥ 70 + 과열점수 50~70:
→ 소량 매수 후 분할 접근
→ 과열 해소 시 비중 확대
```

### 4.4 종합점수 타이밍 축과의 관계

```python
# 종합점수 = 성장(25%) + 안정(35%) + 가격(30%) + 타이밍(15%)

# 타이밍 축 (15%) 내부:
_base = 비과열_rank × 0.30    # 100 - 과열점수
      + 돌파신호_rank × 0.35
      + RS등급_rank  × 0.20
      + 수급강도_rank × 0.15

_decel = -15 if 이익감속경고 == 1 else 0
타이밍축 = clip(_base + _decel, 0, 100)
```

타이밍은 전체 종합점수의 15%이지만, 과열 감도를 통해 **각 전략 점수 자체에도 영향**을 미치므로 실질적 영향력은 15% 이상입니다.

### 4.5 과열/돌파 지표의 한계와 보완

| 한계 | 설명 | 보완 |
|------|------|------|
| 강한 추세에서 과열 지속 | 진짜 주도주는 과열 상태에서도 계속 상승 | 주도주 전략 감도 0.15로 최소화 |
| VCP 실패 | 패턴 형성 후에도 돌파 실패 가능 | 손절 기준 병행 (MA20 하향 이탈 시) |
| 후행 지표 | 과열/돌파 모두 과거 데이터 기반 | 이익가속 등 선행 지표와 조합 |
| 구간 한정 | 약세장에서는 과열 점수 전반적으로 낮음 | 시장 국면 별도 고려 필요 |
