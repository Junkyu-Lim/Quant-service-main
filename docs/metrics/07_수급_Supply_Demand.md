## 1. 수급강도

### 정의
**Supply Strength** — 외국인과 기관의 20일간 누적 순매수가 시가총액에서 차지하는 비율입니다. 종목으로의 자금 유입 강도를 측정합니다.

### 산출 공식

```
수급강도(%) = ((외국인 20일 순매수 + 기관 20일 순매수) × 종가 / 시가총액) × 100
```

| 구성요소 | 설명 |
|---------|------|
| 외국인 20일 순매수 | 최근 20 거래일간 외국인 순매수 주식수 합계 |
| 기관 20일 순매수 | 최근 20 거래일간 기관 순매수 주식수 합계 |
| × 종가 | 주식수 → 금액 환산 |
| / 시가총액 | 종목 규모 대비 비율화 |

### 해석 기준

| 수급강도 | 해석 |
|---------|------|
| > +2.0% | 매우 강한 순유입 (적극적 매집) |
| +0.5% ~ +2.0% | 양호한 순유입 |
| -0.5% ~ +0.5% | 중립 |
| -2.0% ~ -0.5% | 순유출 (매도 우위) |
| < -2.0% | 강한 순유출 (이탈) |

### 시스템 내 활용
- **주도주 전략**: 수급강도 비중 20%, 수급강도 > 0 하드 필터
- **돌파신호 수급 유입**: 수급강도 > 0 시 최대 20점

## 2. 외국인/기관 순매수

### 정의
20일간 외국인과 기관의 순매수를 금액으로 환산한 값입니다.

### 산출 공식

```
외국인 순매수 (20일) = Σ(외국인 일별 순매수 주식수) × 종가
기관 순매수 (20일) = Σ(기관 일별 순매수 주식수) × 종가
```

### 데이터 소스
- 네이버 금융 (Naver Finance) frgn.nhn 페이지 크롤링
- ThreadPoolExecutor 4 workers로 병렬 수집
- 외국인순매수, 기관순매수, 개인순매수 데이터

### 해석 포인트
- **외국인 + 기관 동시 매수** = 가장 강력한 수급 신호
- **외국인 매수 + 기관 매도** = 글로벌 시각과 국내 시각 차이 (방향 불확실)
- **외국인 매도 + 기관 매도** = 양대 세력 이탈 (하락 위험)
- **개인만 매수** = 개인 추격매수 (보통 부정적 신호)

## 3. 스마트머니 승률

### 정의
**Smart Money Win Rate** — 최근 20 거래일 중 외국인 또는 기관이 순매수한 날의 비율입니다. 매집의 **일관성**을 측정합니다.

### 산출 공식

```
스마트머니 승률 = 매수일수 / 20일

매수일: (외국인순매수 > 0) OR (기관순매수 > 0) 인 날
```

### 해석 기준

| 승률 | 해석 |
|------|------|
| ≥ 80% | 매우 강한 매집 (16일 이상 매수) |
| 60% ~ 80% | 강한 매집 (12~16일 매수) |
| 50% ~ 60% | 약간 매수 우위 |
| 40% ~ 50% | 중립 (매수/매도 균형) |
| < 40% | 매도 우위 (이탈 중) |

### 시스템 내 활용
- **턴어라운드 전략 점수**: 스마트머니 승률 비중 15%
- **VCP 신호**: 승률 ≥ 60% 필수 조건
- **돌파신호 수급 유입**: 승률 50%→0점, 70%→17.5점, 90%→35점 (최대 35점)

## 4. 쌍매수 비율

### 정의
**Dual Buying Ratio** — 최근 20 거래일 중 외국인과 기관이 **동시에** 순매수한 날의 비율입니다. 스마트머니의 **합의(Consensus)** 수준을 측정합니다.

### 산출 공식

```
쌍매수 비율 = 쌍매수일수 / 20일

쌍매수일: (외국인순매수 > 0) AND (기관순매수 > 0) 인 날
```

### 해석 기준

| 쌍매수 비율 | 해석 |
|-----------|------|
| ≥ 50% | 매우 강한 합의 (10일 이상 동시 매수) |
| 30% ~ 50% | 양호한 합의 |
| 10% ~ 30% | 약한 합의 (간헐적 동시 매수) |
| < 10% | 합의 없음 (외국인/기관 방향 불일치) |

### 스마트머니 승률 vs 쌍매수 비율

| 승률 | 쌍매수 | 해석 |
|------|--------|------|
| 높음 | 높음 | **최강 수급** — 외국인+기관 모두 적극 매수 |
| 높음 | 낮음 | 한쪽(외국인 or 기관)만 집중 매수, 다른 쪽은 관망 |
| 낮음 | 낮음 | 수급 약세 — 매수 동력 부족 |

### 시스템 내 활용
- **돌파신호 수급 유입**: 쌍매수 비율 20%→0점, 50%→25점 (최대 25점)

## 5. VCP 신호 (변동성 수축 패턴)

### 정의
**Volatility Contraction Pattern** — 마크 미너비니(Mark Minervini)의 트레이딩 전략에서 핵심 개념으로, 주가가 좁은 범위로 수렴하면서 거래량이 줄어들고, 스마트머니가 조용히 매집하는 패턴입니다.

### 발동 조건 (3가지 모두 충족 필요)

```
VCP_Signal = 1  if ALL of:
  ├── 조건1: 가격 CV(20일) < 가격 CV(60일)
  │          └── 가격 변동계수 수축 → 주가 범위 좁아짐
  │
  ├── 조건2: 거래량 평균(20일) < 거래량 평균(60일)
  │          └── 거래량 감소 → 매물 소화 완료
  │
  └── 조건3: 스마트머니 승률 ≥ 60%
             └── 기관/외국인 매집 중 → 스마트머니 확인
```

| 구성요소 | 설명 |
|---------|------|
| CV (변동계수) | 표준편차 / 평균 (상대적 변동성) |
| 가격 CV 수축 | 20일 변동 < 60일 변동 → 안정화 |
| 거래량 감소 | 무관심 or 매물 소화 |
| 스마트머니 매집 | 조용히 사는 중 (핵심!) |

### VCP의 투자 논리

```
[VCP 패턴의 시장 심리학]

1. 이전 상승 후 조정 시작
   └── 초기 매도 압력 (이익 실현)

2. 변동성 수축 (가격 범위 좁아짐)
   └── 매도자 소진, 매수자 관망

3. 거래량 감소 (유동 물량 줄어듦)
   └── 팔 사람은 다 팔았음

4. 스마트머니 매집 (외국인/기관 꾸준히 매수)
   └── 물량 확보, 돌파 준비

5. 돌파! (거래량 동반 상승)
   └── 매물벽 돌파, 새로운 상승 시작
```

### 시스템 내 활용
- **돌파신호 가격패턴**: VCP 신호 ×40점 (최대 40점)

## 6. 실전 활용 가이드

### 6.1 수급 분석 프레임워크

```
[수급 분석 순서]

1단계: 수급강도로 전체 방향 확인
  ├── 양수 → 자금 유입 (긍정적)
  └── 음수 → 자금 유출 (부정적)

2단계: 스마트머니 승률로 일관성 확인
  ├── ≥ 60% → 일관된 매수 (신뢰도 높음)
  └── < 50% → 산발적 (신뢰도 낮음)

3단계: 쌍매수 비율로 합의 수준 확인
  ├── ≥ 30% → 외국인+기관 합의 (강력)
  └── < 10% → 한쪽만 매수 (불확실)

4단계: VCP 신호로 돌파 준비 확인
  ├── VCP = 1 → 돌파 임박 가능
  └── VCP = 0 → 아직 패턴 미완성
```

### 6.2 수급 × 기술적 지표 조합

| 수급 신호 | 기술적 상태 | 전략 |
|----------|-----------|------|
| 수급강도↑ + 승률≥70% | RS등급≥80 + RSI 50~70 | **적극 매수** — 모든 조건 완벽 |
| 수급강도↑ + 승률≥60% | VCP = 1 | **돌파 매수 준비** — 돌파 시 진입 |
| 수급강도↑ | RSI > 80 + MA20 괴리↑ | **주의** — 수급은 좋지만 과열 |
| 수급강도↓ + 승률<40% | RS등급 하락 | **회피** — 스마트머니 이탈 |

### 6.3 수급 × 펀더멘탈 조합

| 수급 | 펀더멘탈 | 해석 |
|------|---------|------|
| 강한 매집 | 이익 가속 + ROIC↑ | **최우선 매수** — 스마트머니가 실적 개선 선반영 |
| 강한 매집 | 턴어라운드 + GPM↑ | **턴어라운드 확인** — 기관이 전환 인정 |
| 약한 수급 | F-Score≥7 + 저PBR | **잠재 후보** — 아직 시장 발견 전 |
| 강한 매도 | 이익 감속 + FCF↓ | **절대 회피** — 스마트머니가 먼저 이탈 |

### 6.4 수급 데이터의 한계

| 한계 | 설명 | 대응 |
|------|------|------|
| 시차 | 순매수 데이터는 T+1일 또는 그 이후 집계 | 최신성 한계 인지 |
| 차익거래 포함 | 선물/옵션 헷지 물량이 현물 순매수에 포함 | 개별 종목보다 지수 ETF에서 더 왜곡 |
| 기관 분류 모호 | 연기금, 자산운용, 증권사 구분 없이 합산 | 세부 주체별 분석 별도 필요 |
| 단기 노이즈 | 1~2일 대량 거래는 블록딜 등 일회성 가능 | 20일 기준으로 평활화하여 해결 |

### 6.5 전략별 수급 활용

```
[주도주 전략]
├── 수급강도 > 0 (하드 필터) — 자금 유입 필수
├── 수급강도 비중: 20%
└── 의미: 기관/외국인이 뒷받침하는 종목만 주도주로 인정

[턴어라운드 전략]
├── 스마트머니 승률 비중: 15%
└── 의미: 턴어라운드의 신뢰성을 수급으로 검증

[돌파신호 (전 전략)]
├── 승률: 최대 35점
├── 쌍매수: 최대 25점
├── 수급강도: 최대 20점
├── 거래대금 변화: 최대 20점
└── 수급 유입 총 비중: 30% (4개 축 중 최대)
```
