# F-Score (재무건전성 종합점수)

> 조셉 피오트로스키(Joseph Piotroski) 교수가 개발한 9점 만점의 재무건전성 스코어링 시스템입니다. 저PBR 종목 중 재무적으로 건전한 기업을 선별하기 위해 고안되었습니다.

## 1. F-Score 개요

### 기본 구조

```
F-Score = F1 + F2 + F3 + F4 + F5 + F6 + F7 + F8 + F9
범위: 0점 (최악) ~ 9점 (최상)
각 팩터: 0 또는 1 (Pass/Fail)
```

### 세 가지 영역

| 영역 | 팩터 | 측정 내용 |
|------|------|----------|
| **수익성** | F1, F2, F3, F4 | 이익 창출 능력과 질 |
| **재무 안정성** | F5, F6, F7 | 부채, 유동성, 희석 |
| **운영 효율성** | F8, F9 | 마진과 자산 활용 |

## 2. 수익성 팩터 (F1~F4)

### F1: 순이익 양수 여부

```
F1 = 1  if TTM 순이익 > 0
F1 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 기업이 흑자 상태 — 기본적인 수익 창출 능력 보유 |
| Fail (0) | 적자 상태 — 근본적인 사업 문제 가능 |

### F2: 영업현금흐름 양수 여부

```
F2 = 1  if TTM 영업CF > 0
F2 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 본업에서 현금이 유입되고 있음 |
| Fail (0) | 본업에서 현금이 유출됨 — 지속가능성 의문 |

### F3: ROA 개선 여부

```
ROA = 순이익 / 총자산
F3 = 1  if ROA_최근 > ROA_전년
F3 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 자산 대비 수익성이 개선 중 |
| Fail (0) | 자산 활용 효율 악화 |

### F4: 이익의 질

```
F4 = 1  if (TTM 영업CF > TTM 순이익) AND (F1 = 1)
F4 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 순이익보다 현금흐름이 큼 → 이익이 현금으로 뒷받침됨 |
| Fail (0) | 회계적 이익은 있지만 현금 부족 → 이익의 질 낮음 |

**핵심 인사이트**: F4는 **발생주의 이익(Accruals)**을 걸러내는 가장 중요한 팩터입니다. 영업CF > 순이익이면 감가상각 등 비현금비용이 커서 실제 현금 창출이 더 많다는 의미입니다.

## 3. 재무 안정성 팩터 (F5~F7)

### F5: 레버리지 감소

```
부채비율 = 총부채 / 자본총계
F5 = 1  if 부채비율_최근 < 부채비율_전년
F5 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 부채 부담 감소 — 재무 안정성 개선 |
| Fail (0) | 부채 증가 — 금융 비용 상승 위험 |

### F6: 유동성 개선

```
유동비율 = 유동자산 / 유동부채
F6 = 1  if 유동비율_최근 > 유동비율_전년
F6 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 단기 지급능력 개선 |
| Fail (0) | 단기 유동성 악화 |

### F7: 주식 희석 없음

```
F7 = 1  if 최근_상장주식수 ≤ 전년_상장주식수
F7 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 유상증자 없음 — 기존 주주 지분 보전 (자사주 매입이면 더 좋음) |
| Fail (0) | 유상증자 실시 — 주당 가치 희석 |

**주의**: 주식분할은 주식수 증가이지만 가치 희석이 아닙니다. 다만 본 시스템에서는 단순 주식수 비교이므로 분할 시 F7=0이 될 수 있습니다.

## 4. 운영 효율성 팩터 (F8~F9)

### F8: 매출총이익률(GPM) 개선

```
GPM = 매출총이익 / 매출액
F8 = 1  if GPM_최근 > GPM_전년
F8 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 원가 경쟁력 향상 또는 제품 가격 인상 성공 |
| Fail (0) | 원가 부담 증가 또는 가격 경쟁 심화 |

### F9: 자산회전율 개선

```
자산회전율 = 매출액 / 총자산
F9 = 1  if 자산회전율_최근 > 자산회전율_전년
F9 = 0  otherwise
```

| 의미 | 설명 |
|------|------|
| Pass (1) | 자산을 더 효율적으로 활용하여 매출 창출 |
| Fail (0) | 자산 효율 악화 (과잉 투자, 매출 감소 등) |

## 5. 종합 F-Score 해석

### 등급 분류

| F-Score | 등급 | 해석 | 투자 판단 |
|---------|------|------|----------|
| 8~9 | **A (최우수)** | 거의 모든 재무 지표 양호 | 강력 매수 후보 |
| 6~7 | **B (우수)** | 대부분 양호, 일부 개선 필요 | 매수 고려 |
| 4~5 | **C (보통)** | 혼재, 방향성 불확실 | 선별적 접근 |
| 2~3 | **D (부진)** | 대부분 부정적 | 회피 권장 |
| 0~1 | **F (위험)** | 거의 모든 지표 부정적 | 회피 |

### 학술 연구 결과
피오트로스키의 원 논문(2000)에 따르면:
- F-Score 8~9점 종목: 연평균 **+23%** 수익률
- F-Score 0~1점 종목: 연평균 **-9.6%** 수익률
- 고F-Score 매수 + 저F-Score 공매도 전략: 연평균 **+32.6%**

## 6. 실전 활용 가이드

### 6.1 F-Score 활용 시나리오

#### 시나리오 A: 가치투자 필터

```
[저PBR + 고F-Score 전략]
1단계: PBR < 1.0 종목 선별
2단계: F-Score ≥ 7 필터 적용
3단계: ROE, ROIC 추가 검증
결과: "싸고 좋은" 기업만 남김
```

#### 시나리오 B: 위험 회피 필터

```
[모든 전략에 F-Score 최소 기준 적용]
- 우량가치 전략: F-Score ≥ 5 (Track A) / ≥ 4 (Track B 금융)
- 현금배당 전략: F-Score 점수 비중 10%
- 고성장 전략: F-Score 직접 사용하지 않지만 영업CF > 0 필터로 간접 적용
```

### 6.2 F-Score 팩터별 중요도

실전에서 **가장 중요한 팩터**는 다음 순서입니다:

| 순위 | 팩터 | 이유 |
|------|------|------|
| 1 | **F4 (이익의 질)** | 분식회계/저질 이익 필터링의 핵심 |
| 2 | **F2 (영업CF 양수)** | 현금이 없으면 기업이 유지될 수 없음 |
| 3 | **F5 (레버리지 감소)** | 금리 상승기에 특히 중요 |
| 4 | **F8 (GPM 개선)** | 근본적 경쟁력 변화의 선행 지표 |
| 5 | **F1 (순이익 양수)** | 기본 수익 창출 확인 |

### 6.3 F-Score의 한계

| 한계 | 설명 | 보완 방법 |
|------|------|----------|
| 과거 데이터 기반 | 과거 1년 비교만 반영 | CAGR, YoY 성장률로 추세 보완 |
| 이진(0/1) 판단 | 미세한 차이도 Pass/Fail | 퍼센타일 랭킹으로 연속적 점수 보완 |
| 업종 무차별 | 금융/제조 같은 기준 적용 | 본 시스템은 금융주 Track B로 부분 보완 |
| 성장 미반영 | 성장률 팩터 없음 | 성장 지표와 복합 분석 필요 |

### 6.4 시스템 내 F-Score 활용 매핑

```
[전략별 F-Score 활용]

우량가치 전략:
├── Track A (일반): F-Score ≥ 5 (하드 필터)
├── Track B (금융): F-Score ≥ 4 (하드 필터, 금융 특성 감안)
└── 점수 내 비중: 20%

현금배당 전략:
└── 점수 내 비중: 10%

돌파신호:
├── F-Score ≥ 7 → +20점
├── F-Score ≥ 5 → +10점
└── F-Score < 5 → +0점

종합점수 (직접 미사용):
└── 안정 축 내 간접 반영 (ROE, FCF Yield 통해)
```

### 6.5 F-Score + 다른 지표 조합 전략

| 조합 | 기대 효과 |
|------|----------|
| F-Score ≥ 7 + PBR < 0.7 | 저평가 우량주 (가치 투자 핵심) |
| F-Score ≥ 7 + ROE > 15% | 고수익 우량주 (버핏 스타일) |
| F-Score ≥ 6 + ROIC > 10% + PEG < 1.2 | 합리적 가격의 품질 성장주 (GARP) |
| F-Score ≥ 5 + 턴어라운드 = 1 | 질적 전환 확인된 턴어라운드 |
| F-Score ≥ 6 + 배당수익률 > 3% | 안전한 배당주 |
